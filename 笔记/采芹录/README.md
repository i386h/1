　　钦定四库全书　　　　　子部十
　　采芹録　　　　　　　　杂家类三【杂説之属】提要
　　【臣】等谨案采芹録四巻明徐三重撰三重有余言已着録是编第一卷论养民教民第二卷三卷多论学校贡举政事利第四巻多论明代人物臧否大抵皆考稽典故究悉物情而持论率皆平允无激烈偏僻之见亦无恩怨毁誉之私胜明人所作诸説部动涉巵言三重所作他语録借周子之一言遂太极隂阳连篇累牍讲学于天地之外惟力主均田限田之议反覆引据持之最坚究而论之自阡陌既开以后田业于民不授于官二千年于兹矣虽有圣帝明王断不能一旦举天下之民夺其所有益其所无而均之亦断不能举天下之田清厘其此在限外此在限内此可聴其买卖此不可聴其买卖而限之使黠豪反得隠蔽为奸滑胥反得挟持渔利而闾里愚懦纷纷然日受其扰故汉董仲舒北魏李安世唐陆防牛僧儒宋留正谢方叔元陈天麟皆反覆言之而卒不能行此犹可曰权不属时不可也宋承五季凋残之后宋髙宗当南渡草创之初以天子之尊决意行之亦终无效则三重所言其迂而寡当可见矣然而论漕粟则驳邱濬海运之非论养兵则驳徐阶塞外不可屯田之谬皆卓然明论其他亦多笃实近理切于事情犹可谓留心经世之学者也乾隆四十六年五月恭校上

　　钦定四库全书
　　采芹録卷一
　　明　徐三重　撰
　　牧民之职当以加惠孤穷为首务盖不能自食之民悬命于上之人者所当约已劳身图其生事者也次当以恤灾为念盖天灾无时一方终嵗无望之人流移破散死亡所系拯救之防不但临事多方尤当先期预备者也次则均田土戸役以抑豪右而扶齐民禁淫奢浮冗以节用度而端俗尚此古人治安本义常日行之以渐图富庶实效者也至于贪猾吏卒藏蠧公庭煽殃乡曲上阏清明之化下乱宁一之规所当简其人数信其约法令公私之情不隔于耳目淳和之俗毋琢于姧欺精察曲防正当加诸此徒不宜苟且听狥者也乃若持廉一节欲行绳吏当先自盟洁已庶可格人无愧乃能督下此寔诗书之士所称诵法圣贤不但名位之流仅欲兢持物论而已其他寛赋缓刑训义重本恒若家庭之事并闗切身之情如此纵未便侈口王仁一郡一境有不实受其泽者鲜矣
　　洪武初令天下郡县立养济院收养孤独残病不能自生之人人月给米三斗柴三十斤冬夏布一疋小口三之一慈悯元元直同父子骨肉矣夫孤独残病饥寒之人上闗九重深念所以图其生理者罔不周详宁忍令海内丁夫壮女不安生事以至流亡失所知必无矣我祖宗徳泽下迨庶品何减乾坤雨露但遐深曲隐正頼长吏奉行若令藏于法之徒美实丧于文之虚具则大非矣夫国家不爱课税征求之额以全无告困穷残命而中间稍有防出无论此升合脂膏有失救死存生之本意倘实政一化为薮则必有万分不能自生者又苦于不可得收而哀号稿死何由知之此在官司所宜推广徳心体隐询瘼而不得目为案牍故事一切付之廷吏已也呜呼士大夫所学何事四民之政童而诵之况国家懿典明章皎然方册当其任者乌得忽诸
　　四民无告王政所最先洪武条章载列监司守令职事更为殷切乃今号啼载涂当事者身逢目覩恬焉靡有诘其情状为之收恤毋论吾心为忍即吾职亦重慢国宪所学所任胥负之矣呜呼傥谓人众不能尽养今郡邑冗耗费何止一端在世途则难已在国典则不经若节此施彼所活何限谓嵗办不能额外夫天下以富养贫天道人理然也在明白正大随宜曲处焉耳我无防防私而以痛闵元元之意明布一境人有不同此心乎如是财固可多方画也谓情伪不易得实夫上以诚感未有不以诚应法以仁立未有不以严行里图街坊乌容尽欺一时蒙眬岂能终掩且既欲处之得所宁不审之周详但使吾有实心终不令虚文冒滥也谓流乞势难兼收夫宇宙同生宁分彼此或遂令寄命或使得还乡各因其情与之生路但患吾心不竭毋患吾力不周也夫万物为已斯道本义一夫颠连痛切吾躬凡有斯人之责皆不可无此意而称于天下曰士大夫也若鲜珍自奉而腹者哀号泥涂渺然乃若不闻高倨快心而黔瘠者宛转谿壑漠然不加动色此亦非人情矣
　　古今救荒惟蠲赈二事余不获尽覩天下地土知其利害姑以身所经行耳目闻见窃谓长淮以北赈急于蠲大江以南蠲重于赈淮北居人星落地旷土瘠粮课甚薄田无畜泄之利一遭水旱千里极目即不蠲亦无所得徴而流乞之民旦暮悬命沟壑之外便虞剽掠是赈之惠实于蠲也大江以南田赋浩繁生人骈塞无借之人居多升斗之给难遍若复以薄收之物充重额之输则地方之存无几而食货之直必涌且饥寒交廹敲朴难堪斥卖流亡势所不免是蠲之益重于赈也夫蠲赈二事荒嵗何可废一顾于并行之中畧计其虚实重轻则然耳然江南田多钜室蠲亦难从防例列为差等明示低昻亦厚业者之所不能怨也至于赈之一事苐责成良有司兴发劝募公勤以行之多方以稽之务使无业者得济必死者得全斯为实政夫发帑遣官岂非圣朝徳意苐奉行乖方反滋骚动列郡之费不赀民间竟无大益此亦当事者所宜鍳徃觉来先图无而后可言其利也
　　为郡守积谷备荒是第一务不入私室不为馈遗买谷入仓择部民有行父老主之仓须完缮高燥用柴盖防待主仓者如家之主计优以礼遇免其杂役不使左右羣小有所需求毎厫有定数折耗有罚再勤加检问察其盈缩一遇凶歉发而平粜豫访耆徳散布各乡质盟于明神感动以诚恳间遣自爱属吏廵行稽查或亲乘小舟突诣亷问夫预备则有待无弊久而益饶散粜则民便利均不至羣聚而又身劳诚感不废稽防庶几民有实惠如此而百姓犹流亡丧乱始可委诸天矣
　　晦翁常论常平仓弊云米俱浮埃空殻不敢挑动又影射侵支有难理会愚谓此等事正生治人实希法从人弊故耳今谈者亦苦口常平之难行谓害甚于利夫居位任职使国家良法无利而滋弊不知士大夫有生于世其处身利物当如何而漫无用心如此也
　　救荒一事士大夫稍以民物为懐未有不恻然图其实济苐心虽欲尽而势所不遂身所难亲一人之心思耳目有限而闾阎不悉之情隐羣小乘便之姧欺不无此顾而彼遗名然而实否意美而法弊费实而惠虚纵有禹稷饥溺之思无禆沟壑颠连之苦防人乃言荒防无竒诚恨之也偶见世庙时广东佥事林希元上荒政樷书内列纲六目二十有三曰有二难得人难审户难有三便极贫之民便赈米次贫之民便赈钱稍贫之民便赈贷有六急垂死贫民急饘粥疾病贫民急医药病起贫民急汤米既死贫民急墓瘗遗弃小儿急收养轻重囚繋急寛恤有三权借官钱以粜籴兴工作以助赈贷牛种以通变有六禁禁侵渔禁攘盗禁遏籴禁抑价禁宰牛禁度僧有三戒戒迟缓戒拘文戒遣使其审势立欵推情设防可谓几尽若斟酌事宜务令实惠及民则在仁人君子自尽厥心焉耳窃思遣使之戒厥有前防徃年江南被灾发帑命官赈济无益而更滋骚扰防司马文正公谓不如专任监司守宰者真为石画此在朝廷之上老成之识但为责成核实而不牵惑于少年浮论则善矣若监司守宰身任其事又以得人为急而难葢苦心综理虽在我而布散分任须用得其人茍一人失用即一处一事为其所误而僵死不知几何人矣防富郑公在青州所用官吏自前资待缺寄居者皆赋以禄使就民所聚选老弱病废者廪之公感之以诚人人尽力今固无待缺寄居惟有见任本境官吏虽当民艰之日同有良心顾地方事任孔棘如欲付孤子于乳母所以审择之伺察之亦不得不加之意也丘文庄则欲于所部致仕闲住待选依亲等项官吏监生与夫僧道耆老医卜人等凡平日为乡人所信服者俾各就所在因便给散是用本乡士民也然又必为人所信服亦岂得一槩付之天要各就一方人情土俗酌于才品而期其不负委任若官吏士民正不拘耳诚幸得慷慨盟天矢日与吾共志之人而即属以辨户业之上下有无察丁口之老弱疾病孤苦或随居给散以便民或就食其不能出入徃来者令逺近饥寒赤子宛然若咸当吾前而凡荒政所施徳意靡不若吾躬行而人实受之岂非一方生灵厚幸欤至于牛种之贷宰牛度僧之禁则预为荒后生事计者又不但茍舒今日一朝之急惟预备劝助二端不在明白开欵盖劝助一事行之不善反成苛索预备一节当在平日临廹则难倘被灾太甚应用浩繁仓卒措置钱米不免巧妇束手之叹若只倚办官库额少用浮中道称竭安得委而坐视欤念此必当先事不忘经营临事广为方畧凡诸行仁之事以身倡之则好义者自众不可强廹以不堪及民之政以实致之则受惠者必得母令借窦以生弊要在贤明并济经权一体上下倘按常守矩而尽其在我或用智多事而不通物情利益未施怨咨横出矣嗟嗟志在苍生者必不自专一人之见则于艰难倥之秋真如一家一身之厄焦劳之外更务周询凡所以集众思借羣力察幽细行实然者宜更如何皇皇切切也
　　窃忆防年灾荒官司出米煑粥然食粥者死甚众非粥之能死人盖饥寒雨湿人自无生理耳当知得食而生是粥之功饿伤疫染而死非粥之过倘归过于粥而不为料理则可生者亦死是与于不仁甚也惟官司出米或委任所属官吏或差派地方富民中间苟且塞责又不无侵尅减省襍以他物乃足为害闻有和入石膏沙土者乆困濒死之人膓胃岂堪着此是适以好生行杀也若择人而任用意稽察当不至是食粥之人其间亦有自能食而冒滥支给者似所当裁已给粥而有室可归者亦不必虑惟流集饥人聚守于此腹待食以次逓给日晏得餐数嵗丰穰饕恣之人仅仅得此已不能饫且老防疾病羣居一厰疫气相袭而又当隂两弥月湿衣冷地俱足伤人此官司之所无可奈何者也夫丁壮滥食非曰恶之特分取无告之口且令官司势不能遍亦为未平饥人不下千百林立鱼次取给鼎镬之间岂能如期得食米珠薪桂欲令充饱其势自难而逺村境外扶老提弱坐卧荒凉浮露之厰防穿下透渍骨侵肌以积羸之人备六气之贼疾病死亡又岂粥糜所能救疗呜呼羣公皆才名良吏犹然患此岂救荒真无防耶窃议欲仿范文正营作之意令丁壮者以力得钱不虚坐食老弱则编名定额限米分炊数止贫廹给可从优而散处空闲各有覆借至于疾病医药自不可无脱有死亡瘗理亦不容己若预备实积以奉圣朝徳意勤亷自矢以忧百万生灵此亦吾辈立心之事也呜呼行之惟艰谈何容易捉麈而论兵得无为介胄嗤乎
