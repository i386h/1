玉烛宝典　(隋)杜台卿撰　日本前田家藏旧钞卷子本　日本尊经阁文库藏
　　杜台卿，字少山，博陵曲阳人也。父弼，齐卫尉卿。台卿少好学，博览书记，解属文。仕齐奉朝请，历司空西合祭酒、司徒户曹、著作郎、中书黄门侍郎。性儒素，每以雅道自居。及周武帝平齐，归于乡里，以《礼记》、《春秋》讲授子弟。开皇初，被征入朝。台卿尝采《月令》，触类而广之，为书名《玉烛宝典》十二卷。至是奏之，赐绢二百匹。台卿患聋，不堪吏职，请修国史。上许之，拜著作郎。十四年，上表请致仕，敕以本官还第。数载，终于家。有集十五卷，撰《齐记》二十卷，并行于世。无子。有兄蕤，学业不如台卿，而干局过之。仕至开州刺史。子公赡，少好学，有家风，卒于安阳令。公赡子之松，大业中，为起居舍人。

　　东风解冻蛰虫始震鱼上负氷獭祭鱼鸿鴈来 【此时鱼肥美獭将食之先以祭也鴈自南方来将北反其居高诱曰是月之时鲤应阳而动上负氷也今案庄子曰潜鲠春日毁滴而盖衢者鳝也司马彪注云潜水中也鲠涩滴池盖辞衢道也言冬日氷鲠涩不通春日微温□池氷而为道者也鳝鱼也是则鳝鱼亦贠而出祭鲤盖取其尤好者人淮南子曰獭知水之高下高诱注云高下犹深浅萧广济孝子将曰獭水兽也似狗而痹脚青黑色立春则羣捕鱼聚其所获陈列于地一纵一横对之而伏也】 天子居青阳左个桑鸾辂驾仓龙载青□衣青衣服仓玉食麦与羊其器疏以达 【皆所以慎时气也青阳左个大寝东堂北偏也鸾辂有虞氏之车也有鸾和之莭而餝之以青取其名耳麦实有孚甲尸?禹木羊畜也时尚寒食之以安性也器疏者刻镂之象物当贯士而出也高诱曰东向堂故曰青阳北头室故曰左个个犹隔也今案礼下曹君天下曰天子郑玄注云天下谓外及四海也今汉于蛮夷称天子于王侯称皇帝周书太子晋篇曰善至于四海四夷四荒其表又曰四荒皆至莫有恐呰乃豋为帝注云呰叹恨也舍五等之尊卑更论事义以为之名孝经援神契曰天覆地载谓之天子易干凿度曰天子爵号也天子者继天理物改正统一各得其宜父母天地以养民至尊之号之也】 
　　是月也以立春先立春三日大史谒之天子曰某日春盛德在木天子乃齐立春之日天子亲帅三公九卿诸侯大夫以迎春于东郊还反赏公卿诸侯大夫于朝 【迎春祭仓帝灵威仰于东郊之非也王居明堂礼曰出十五里迎岁殷也周迎郊五十里】 命相而德和令行庆施惠下及非民 【相谓三公相王之事】 庆赐遂行毋有不当乃命太史守典奉法司天日月星辰之行宿离不忒无共经纪以初为常 【典六典法八法离读如俪偶之俪谓其属冯相氏保章氏掌天文者也其相兴宿偶当审候司不得过差今案春秋说题辞曰天之为言镇也居高理下为人君阳精也合为太一分为殊名故立字一大为天说文曰天巅也主高旡上从显也青徐以舌头言之文理也然高远也又谓之玄县也如悬物如上也白虎通曰天者身也镇也男女惣名为人天地无惣名何天贠地方不相类也杨泉物理论曰天者旋也积阳纯刚其体四旋而洞达其德清明而车圴羣生之所天仰故称之曰天正历曰天者远不可极望之雰然以玄为色其人大无不苞裹其动靡有烋息谓之天者一大之名也礼统曰天之为言镇也神也陈也王??也施生为物本连转精神功郊布而陈其道可王??重谓也】 天子乃以元日祈谷于上帝 【谓以上辛郊祭天也】 乃择元辰天子亲载耒 【音虑猥反】 秬棤之于叅保介之间师三公九卿诸侯大夫躬耕帝藉天子三椎 【勅雷反】 三公五推卿诸侯九推 【元辰盖郊后吉辰也未耕之上典者也保介车在也人君之车必使勇士衣甲居右而叅乘备非帝也保介犹衣也介甲也帝藉为天神借民力所治之田也】 反乃执爵于太寝三公九卿诸侯大夫皆御命曰劳酒 【既耕而燕饮以劳群臣】 天气下降地气上腾天地和同车木萌动 【此阳气蒸达可耕之候也农书曰土长冒振陈根可拔耕者急发今案春秋元命苞曰地者易也言养物怀任交易变化合吐应莭故其立字吐力之物一者为地宗均注云地加土以力者言地变化成物功着也加一者奉太一也释名曰地应也其地平载万物也亦言谛也立所生莫不审谛也亦谓川川顺也言顺干之者也】 王命布农事命田舍东郊皆修封疆审端俓术 【田谓田畯主农之官也封疆田首之职分也术周礼作遂小沟也步道曰俓今苍颉篇曰术邑中道也】 盖相兵陵险原隰土地所宜五谷所殖以教导民必躬之亲 【相视之也】 田事既饬先定准直农乃不或 【准直谓封坛俓遂也】 命乐正入学习舞乃修祭典命祀山林川泽檥牲毋用牝 【为伤任生之类】 禁止伐木 【威德所在之也】 毋覆巢煞孩者 【朝来反也】 虫胎犮 【音为老反】 飞鸟母麛母卵 【为伤萌幼之类者高诱曰麋子曰大庶子曰麑今案尔雅曰麋子庆音恶拏反郭璞云江东亦呼鹿子为庆国语鲁语曰兽长麑犮唐固注云亦麋子曰庆又社志志云蜂（虫广）不螫婴儿昬蝱不食犮馰樍然则馰例似称犮庆犮庆麛麑字并两通也】 毋聚大众毋置城郭 【为妨农之始也】 掩骼 【音格】 埋胔 【在赐反为死气逆生也骨枯曰骼内腐曰胔也】 不可以称兵必有天殃 【逆生气也】 兵戎不起不可从我始 【为客不利生人则可】 毋变天之道 【以阴政犯阳也】 毋绝地之理 【易刚柔之宜也】 毋乱人之纪 【仁之时而举义事】 孟春行夏令则雨水不时 【已之气弃之也】 草木蚤落国时有恐 【以水讹相惊也】 行秋令则其民大疫 【甲之气弃之】 焱风异雨惣至 【正月宿直尾莫好风其气逆也曲风为焱案尔雅扶摇谓之焱音方遥反李巡曰扶摇暴风从下升故曰焱焱上孙炎曰四风从下上故曰焱音义曰尸子曰风为颓焱】 藜莠蓬蒿并兴行冬令则水淹为败雪霜大击首种不入 【去之气弃之也旧说云首种谓稷也高诱曰雨霜大击伤害五谷今案释名曰雪娞也水下遇寒而归凝娞娞然下春秋考异邮曰霜之为言亡人物以终释名曰霜丧也其气惨毒物丧之也】
