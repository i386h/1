　　夷坚志　　　　　　　　小説家二【异闻之属】提要
　　【臣】等谨案夷坚志五十卷宋洪迈撰迈所着容斋随笔已别着録是书所记皆神怪之説故以列子夷坚志事为名考列子谓大禹行而见之伯益知而名之夷坚闻而志之正谓珍禽异兽如山海经之迈録仙鬼诸事而名取于斯非其本义然唐义原尉张慎素已有夷坚録之名则迈之名亦有所本也陈振孙书録解题载夷坚志甲至癸二百卷支甲至支癸一百卷三甲至三癸一百卷四甲四乙二十卷共四百二十卷赵与峕宾退録亦载迈夷坚志三十二编凡三十一序不相重各节録其序之大畧颇为详备此本仅存自甲至戊五十卷标题但曰夷坚志而失载其原序证以与峕之所载乃支甲至支戊非其正集惟与峕记支丙作支景谓避其曽祖之嫌名而此仍作丙疑传冩者所改也胡应麟笔丛谓所藏之本有百卷周亮工书影谓应麟所藏乃支甲至三甲此殆即胡氏之本而又佚其半欤朱国祯涌幢小品不知为志中之一集乃云夷坚志本四百二十卷今行者五十一卷盖病其烦芜而删之则悮之甚矣陈振孙讥迈为谬用其心其説颇正张世南防宦纪闻则谓迈欲修国史借此以练习其笔似乎曲为之词然其中诗词之往往可资采録而遗闻事亦多足为劝戒非尽无益于人心者小説一家厯来着録亦何必拘于方隅独为迈书责欤乾隆四十三年五月恭校上

　　张相公夫人
　　钱履道字嘉贞京兆咸阳人也金皇统中游学商虢过鄠县贪程不止独一仆相随天曛黑不复辨路信马行到一大宅扣门将托宿遇小妾从内出惊语之曰此地近山多狼虎岂宜夜涉钱曰适不意迷途敢求栖寓一席之地但不知为何大官第宅妾曰是河中府尹张相公之居相公薨亡唯夫人在须禀命乃可遂入白之少顷延客相见髙堂峻屋明烛盈前已罗列杯盘夫人容色端妍冠服华盛便与同宴侍儿歌舞之妙目所未覩钱自谓竒遇若游仙都情思荡摇莫知身世之所留拱手敬坐不轻交一谈诸人以为野戅相视笑侮罢席就枕俄而烛至夫人者复来众拥之登床钱趋下辞避强之再三于是共寝明旦留之饭钱本漂泊旅人旣称惬怀抱累日不言去一夕正欢饮间闻戸外传呼呵导之声云相公且至夫人遽起诸妾皆奔忙散钱窜伏暗室不敢喘怖因假寐久之狐嘷鸦噪东方旣明人屋俱亡但已身卧于丛古冢耳狼狈而出逢耕夫始得官道衣上余香芬馥经月乃歇
　　楼烦道中妇人
　　岚州宜芳县飞鸯堡村民难言往楼烦县中道少憩逢妇人素衣髙髻年二十许揖而问曰我自楼烦来往岚州独行迷路不知从何为便言指示之时分手妇人长吁一声遂仆地掖视之死矣言就邸舍求汤饮灌救竟不起傍人过者见妇人死不明畏为己累执以告保伍遭系县狱讯治虽自诬云杀而取其赀然僵尸无痕伤又不能供所掠何物郡遣曹掾明生者审究呼问言曰汝实杀人邪对曰难言也明生不悟其姓名再问之复然以为疑乃曰然则所谓难言者非汝邪又对曰然明还郡具白太守别选吏启圹验之但得朽木一片于柩中无从鞫勘因纵释使去言在家事父母极孝谨为乡社所重至是盖获天佑云
　　普光寺僧
　　武城之东普光寺行童元晖近村王氏子也旣作僧为街坊化士嗜酒不检一意狎游年二十五岁得疾甚恶还其家困卧阅一寒暑忽昏不知人举室环泣少顷仰首长鸣顿仆于下问其所苦稍能言曰腰脊之下尾骨痛不可忍呼疡医孔彦璋视之乃短驴尾自皮肤间崛出父畏丑状宣播急掩其衣痛愈切复裸以示人然后止明日长尺许又明日遍体生毛首面已肖驴形数日后蹄鬛俱备两耳翘翘然哮吼悲鸣四支据地卓立俨成真驴家人议欲杀之寺僧云不可此天所以示戒彰其恶报以惩后来如杀之是背天理将为君家不利于是畜于廏中而弗施辔勒驴嘶噉不已且乱齧人试举鞍置前则耸耳以待若有喜意重致远能日行二百里凡十年方死
　　刘将军
　　金兵据齐鲁之地改奉符县为泰安军其皇统二年累月不雨汉儿刘将军为守祷于岳岱不应继致祭龙潭赫日滋炽刘怒命丁夫运石土欲填潭使平夜梦神告曰天久无雨非吾之罪今四海之内凡一勺之水则有神主之吾弗得预又岂敢上违天律輙降膏泽邪幸使君察之也刘寤而愈怒黎明率千余卒益辇石土投寘潭内比暮遂平然到晓复如故刘了不警悟但竭人力而为之当昼隆寒风倐起而雷电从潭出山阜皆震吏民惧甚刘犹督役不已数日间暴卒雨乃沛然
　　淑明殿马
　　完顔亮正隆中泰安守贝实勒始到都欵谒东岳庙遍礼羣祠至淑明寝殿地有流血大惊躬率从吏周行检视见后塑像一指折血淋漉弗止而首饰臂钏及供床黄白器皆亡失即捕典掌者系狱讯治虽加以峻刑终不得其实后因月旦再诣庙备牲币奠享炷香敬祷曰后宫严閟深固讵容穿窬可入今狱久不决必累无辜惟神至聪极明愿显示诛殛以快民意祝罢囘车明日晚阍卒走报云殿西素壁间旧绘马四匹早忽不见山下人尽闻马足响而不覩其形久之各衔一人至一僧一童并奚奴贝实勒急策马往先谢神威然后验问四囚骈械庑下如有物执持者是时有女眞千戸曰阿实达虐而敛所部诛求赀贿其二人无以应命与竹林寺僧同谋自殿屋山翼鑚瓦断椽经旬日始仅得达故外人莫知藏其物于僧所童行与闻之于是具奏于廷亮令杖杀千戸及其两奚奴并窜僧童于远裔
　　王二
　　王二陇州人其居在黑松林跑谷世以畋猎射生为业用是得名因与众逐鹿至深崖迷失道正彷徨次遇女子渡水来少年貌美而身无衣袽视王而笑王平生山行野宿习见物怪虽知为非人殊无惧色咄之曰汝邪怪邪女子又笑而不答良久乃问王曰尔何人王始稍敬异揖而言本山下猎徒今日逐虎失踪致堕兹处生死之分只在须顷愿娘子哀之女曰随我来当示尔归路遂从以行登絶髙巉嵓之峯涉囘环过膝之水涂径荦确足力不能给女不穿屦步武如飞到一洞有大石室境趣防寂如幽人居不闻烟火气寝室尤洁雅王顾傍无他人戏言挑之欣然相就夜则共榻昼则出求果实以啖之居月余王念母乏供养以情泣告女曰我欲暂归徐当复相寻女许诺送出官道乃别王感其意爱他日再访焉试与之语邀同归略不谦拒携手抵家王妻赵氏旣育三男女矣此女又生两子与赵共处甚雍睦逢外客至必惊讶敛避或独走入山经月不返终不火食王亦任其去留后二十年犹存
　　河中西岩龙
　　皇统中河中府大旱太守李金吾祈祷未效闻西岩寺僧慈惠戒律精髙为缁徒所仰乃往请之僧曰身老无以动天地但每日説法之时必有一老叟来听讲莫知所从来疑为龙也当试扣之须金吾明旦至此洁诚以待李曰诺如期叟至李正从此语望其入寺即焚香设席命左右掖之再拜致词叟惊止之曰使君屈膝于山翁敢问何以李曰亢阳为灾五种不入万民将无以生愿龙君慈仁亟下甘泽当肇建祠宇岁时奉祀以彰显大神之威灵惟神念之叟无言少顷颦蹙而叹曰噫泄吾天机者师也吾死无日矣遂告李曰使君勿忧誓以死报又顾僧曰吾今以师故获罪上穹立降诛罚吾即死尸坠于地然不出此境中乞为作证明使阖郡民为行坛七昼夜庶几借此功徳可获超升僧许之而去于是一雨三日外邑虞乡有死龙堕山下李尽率士庶召浮屠千人诣其处筑坛场延慈惠演供事毕龙见于空作人言谢曰吾虽天诛而頼法力救助乘无上妙因得为菩萨龙矣李为建庙请额于朝且名其地为苍龙谷小説载释照讲法华经于嵩山有三叟日来谛听自言是龙照以天旱令降雨叟曰雨禁絶重傥不奉命擅行诟责非细唯孙处士能解弟子之祸照为谒孙思邈致恳是夜千里雨足三叟化为獭匿于孙所居后沼遭使者捕执孙使解而释之事颇相
　　燕王迁都
　　金天徳二年五月以燕山城隘而人众欲广之其东南隅曰通州门西南曰西京门各有髙丘俗呼为燕王不能知其为何代何王也及是立标埒定基址东墓正妨碍议欲前其北面以増雉堞工役未施之数日都民于中夜时闻人声云燕王迁都皆出而观之见銮辂仪衞前后杂遝灯烛荧耀香风袭人罗列十里从东丘至西遂灭明夕复然民以白府留守张君为请于朝廷乃迂枉其垒以避之
　　五郎君
　　河中市人刘庠娶郑氏女以色称庠不能治生贫悴落魄惟日从其侣饮酒郑饥寒寂寞日夕咨怨忽病肌昏冥不知人后虽少愈但独处一室黙坐不语遇庠輙切齿折辱庠郁郁不聊委而远去郑掩关洁身而常常若与私人语家众穴隙潜窥无所视久之庠归舍入房见金帛钱绮盈室问所从得郑曰数月以来每至更深必有一少年来自称五郎君与我寝处诸物皆其所贶不敢隐也庠意虽愤愤然久困于穷冀以小康亦不之责一日白昼此客至值庠在焉翻戒庠无得与妻共处庠惧徙于外馆一听所为且铸金为像晨夕瞻事俄为庠别娶妇庠无子祷客求之遂窃西元帅第九子与为嗣元帅赏募寻索邻人胡生之妻因到庠家见锦绷婴儿疑非市井间所育者具以告帅捕庠及郑械系讯掠而籍其赀狱未决神召会物辟重门直入狱劫取凡同时诸囚悉逸去帅大怒明日复执庠夫妇棰楚苛酷是夜神又夺以归而纵火焚府治楼观草场一空瓦砾砖石如雨而下救火者无一人能前帅无可奈何许敬祀神不复治两人罪五郎君竟据郑氏焉
　　宋中正
　　魏人王员外以纳粟得州助教家富而性狠狼戾恣暴出过神祠未尝加敬或指而詈侮金国正隆初有士人通谒曰宋中正旣延见为纵陈祸福其言似涉讥戒王不悦答之曰天生徳于予祸福其如予何客曰君悖戾愎谏匪朝伊夕荧惑眞君将下临君家速禳之尚可免戾王曰使祸可禳而去则福亦可祷而来子勿以不根之辞诳惑于我客咄咄不已王叱遣之经旬浃又一客绯衣亦称姓宋与王语如中正之辞王曰旬日前有一宋秀才相访意欲相恐胁吾固拒不欲听君岂其党邪吾平生直心于神事无所畏敬君衣朱衣而姓宋得非荧惑之精乎复叱之其人出外仰天大呼即有块火从空飞下众争趋救王犹鸱张大言曰不足救也此不过能庐舍耳俄顷火焰旋转散为数十炬王屋邸无远近一切荡然虽金坚玉白俱成煨烬其居之侧故有火星庙略无所损
　　七娘子
　　大河之流截大行而东注峻滩数十水势湍悍鱼鼈不能停居其一曰七娘子滩山顚有龙女庙山下民千家当夏潦稽天岁有隄防之劳沦垫之虑父老杂议将徙聚落于他所士人韩元翁者老成博雅为党里所信乃往谋焉元翁曰吾曹世世居此坟墓庐舍其传已久一旦委去于心终不安试沥恳于龙祠视其从违乃随事为计亦未晚也于是醵钱具牲牢酒醴择日诣庙求迁其祠于河滨掷杯珓以请得吉卜众拜而归方撰财虑费是夜雷风大作声如頽山暴雨倾河狐啸哭山下人尽起皆以为贻神怒比晓霁色融怡一庙俨在平地尺椽片瓦无有坏隳至于壁泥塑像一切妥贴面势平正基宇坚牢絶胜于旧自是滛涨抵庙岸即止民无复忧
