先进遗风 明 耿定向撰 毛在増补
　　
　　钦定四库全书
　　
　　子部十二
　　
　　先进遗风
　　
　　小说家类一
　　
　　杂事之属
　　
　　提要
　　
　　臣等谨案先进遗风二卷明耿定向撰毛在増补定向有硕辅宝鉴要览已着録在自署太仓人其始末则未详也是书畧仿宋人典刑録之体载明代名臣遗闻琐事大抵严操守砺品行存忠厚者为多盖明自嘉靖以后开国敦龎之气日逺日漓士大夫怙权营贿风尚日偷定向陈先进之懿行以救时弊故所纪多居家行已之细事而朝政罕及焉考其著书之时正分宜骄怙之日定向其有微旨乎其间如曲誉李东阳之类未免乡曲之私提唱姚江之学亦未免门户之见然著书大旨不在是略其小疵可也此本为陈继儒秘籍所刻体例混淆定向原书与毛在所续辑不甚可辨间有论断亦不知为谁语以行欵推之殆每条第一字跳行者为定向之书其第一字平书者为在之书欤今未见定向之原本不可考矣
　　
　　乾隆四十六年四月恭校上

　　先进遗风卷上
　　
　　（明）耿定向 撰　毛在 増补
　　
　　学士宋文宪公濓公尝曰古人为学使心正身修措之行事俯仰无愧而已繁辞复说道之蔽也尤笃于伦品与人交和易任真内诚外恕一见信服汲引后学如恐不及平居布衣疏食无异贫士或劝为子孙计曰富贵一家物哉我乃所以遗之也自少至老未尝一时去书致仕居青萝山闭门纂述人不见其面权要索文者非其人虽置金满槖一字不与纵与之亦不受馈日本使奉勅请文以百金为献公郄不受上以问公公对曰天朝侍从之臣而受小夷金非所以崇国体也
　　
　　参政陶姑孰郡公安上与儒臣论学术公对曰道之不明邪说害之也上曰邪说害道犹美色眩目鲜不为惑自非豪杰不能决去夫邪说不去则正道不兴正道不兴天下乌得而治公顿首曰主诚探本至论
　　
　　惟公当天造矢谟首以正学术为主而髙皇都俞重惩邪说之为惑伏覩教民榜崇正息邪赫然不显矣毋亦公启之耶宋学士称说古人学术必证之心身行事先进之学术可覩已
　　
　　杨文贞公士竒以元宰归省过湖中邂逅一张参政者风驶舟上下则各举手一笑竟别抵维扬郡守令先日候无耗翊日舟至守令竟不相闻过去访友人则相与聨寝语逹旦自常廪外无别供乡人得以只鸡束薪相辞受其细如此其大可知也予因是又知当时上下省事殚力民务而化国日长矣予观文贞为此良有本文贞雅言天下事当以天下万世心处之嗟夫末世之有天下万世之心者谁哉（见胡子衡庐稿）公正统初毎朝罢归邸或正襟危坐长吁不已或独倚阑立或月下闲行通夕不寐夫人问其故不答一日早朝回忻然喜动颜色夫人问曰毎朝回多忧今独喜何也曰主少国疑担荷重任惧不克胜故多忧今早见上聪明已能览章奏决事重任可释矣先辈忧喜一系国事如此（见灼艾集）
　　
　　李文逹贤入仕初公以不识为歉南阳守公友也邀之徃见文达不可曰素无识而造门是求知也士夫两贤之
　　
　　杨文敏公荣天台鲁中丞穆登进士后还乡杜门读书絶迹谒请嗣戒行还京有司具腆赆赠之固辞弗受或曰行以赆礼也奈何拒之鲁公曰筮仕之始未有分毫益于乡里而先厉之忍乎嗣拜监察御史寻擢福建佥事摧抑豪右无所顾避文敏时执政家人有犯者亦不少贷文敏以为贤特荐于宣庙遂拜佥都御史鲁公歴仕二十余年家无余赀被服如寒士卒之日家无以殓赖诸公卿赙始克襄事迹公终始若此于文敏之知可谓亡负矣
　　
　　论曰鲁公初第时即不忍厉其梓里如此其衷笃其识逺哉嗣刘忠宣发解时台司檄有司佐赆公念里中阻饥恳辞之心与鲁公同穷视其所不取逹可知已鲁公佥闽臬时法绳执政之家亦不少贷刚正不阿其自立者素也乃文敏故以为贤特荐之宣庙此与文定之荐范令同达视其所举前辈之为国为民无以有已如此按鲁范两公皆天台人人谓台人多峥嵘有节槩其风自昔已然耶（挽近世俗子一离黉序希觊有司如责偿夙负然既列膴仕其媚权贵如脂韦然识趣若此世何赖焉）
　　
　　杨文定公溥执政时其子自乡来省至京邸公问曰一路守令闻孰贤其子曰儿道出江陵其令殊不贤曰云何曰即待儿茍简甚矣乃天台范理也文定默识之即荐升徳安府知府甚有惠政民到今颂之再擢为贵州左布政使或劝范当致书谢范公曰宰相为朝廷用人非私理也何谢竟不致一书逮后文定卒乃祭而哭之以谢知己云
　　
　　程襄毅信公一见而竒之与论人品确有定见曰逺到器也荐为吏科给事中卒为名臣
　　
　　宪副刘公仁宅华容人忠宣公父也永乐庚子贡士初仕为瑞昌令同邑人严某令高安同入觐文定遣一价往瞷之价还白公曰严丈富贵雅称官也刘丈藁席布被瓦盆煤灶犹然穷人耳公心识之刘与严皆公邻邑人且有■〈女连〉严卖刘特先见贽以币公麾之刘嗣见具茗一袋蜜一缶耳公嘉纳寻擢为御史刘公为御史时六七人共一马更迭出入常与同僚约过除岁各具一肉一蔬或具肉二豆酒一壶同僚甚讶其奢公出所有惟一枯鱼而已后升广西宪副归囊惟七金云正统庚申文定以展墓归里刘公时为御史在京公还朝过华容便造焉问忠宣曰汝父在否曰在道中未回曰汝母安在曰适邻家磨面去矣乃起徧视家中所有遂引忠宣诣寝室见床上惟蒲席布被褥喜曰所操若是可称御史之职矣既去刘公回忠宣白其事刘公曰此必乡先生杨少保也其为人缜宻故观人于所忽若此（右忠宣为萧佥宪聪言见梦醒録萧佥宪河南新野人按文敏过华容忠宣时尚龀）
　　
　　梦醒録曰噫今有官侍御而蒲席布被褥者乎其妻有躬操井臼以为养者乎惟昔东汉王良为大司徒布被瓦器司徒史鲍恢以事过其家良妻布裙曳柴从田中归事类此盖尝叹世之为官者其行谊日薄一日其受用日丰一日其声价日损一日其势焰日张一日而民生则日憔悴一日岂不悲哉余按李文正着刘公状述刘公令瑞昌其平乱功甚巨惠洽于民甚深人有不为也而后可与有为信哉文定初遣仆侦公于旅邸嗣过家又瞷之维时庙堂上急于知人也如此举措如此吏治蒸蒸民生熈熈有以也
　　
　　附余里庄襄刘公天和任三邉总制时差健卒取其孤孙暨一孤侄来任所抚之比至华州其仆夫偶棰门役门役肤愬于州守州守怒弗为礼封锁其门即薪米不供二孤至饥渇甚不得已令从者踰垣窃出乞食于素所知交家微行去州守随投牒公所备陈从仆虐门役状二孤既抵任所环公夫人前泣诉途中为守所苦云云夫人心怜甚嗣州守以事谒制府家众跂足侧窥计公必督过守已乃公故礼遇之有加后复特荐其贤能于朝
　　
　　余里刘端简公采任右司马时余同年尚德恒氏为麻邑令故亷直不避贵势公姻里多不恱在京日譛于公所余一日谒公意为解之以同年故嫌于出口嗫嚅不能尽词公曰然吾故知尚令必洁亷人也不待公言何以知其然凡人有欲则不刚即尚令君之惩治吾家诸仆更不少贷其操可知已余叹公明决而能自克若此时时述为诸公卿道之意凡为乡缙绅者胥识此意庶牧民者得展布云
　　
　　论曰国家张官置吏凡以安民也吾侪诵诗读书师法孔孟将以究安民之术也一旦跻膴仕或进据郡县之上或退居闾左之间惟恱牧民者之曲意徇我骩法庇我而于斯民之瘼若蔑闻者则上负国家自负所学亦甚矣省夫
　　
　　杨文懿公守陈以洗马乞假觐省行次一驿其丞不知其为何官公与之坐而抗礼卒然问曰公职洗马日洗几马公漫应曰勤则多洗懒则少洗无定数也俄而报一御史且至丞乃促令让上舎处之公曰此固宜然待其至而让未晩也比御史至则公门人也跽而起居丞乃睨御史不见蒲伏阶下百状乞怜公卒亦不较归则读书田间不轻谒官府时陆司寇卒有御史过而不吊也公欲警之于其来谒令阍人辞曰适闻吊陆尚书先至彼以候以故失迓御史旋复诣陆先令其子主事君至彼辞曰家大人迟公良乆不至乃归其人颇惭公伟才髙第藻词渊学艺林推为雄长而介洁自持未尝干求恩泽为翰林五品者十有六年而不调权幸有欲引之者使所亲谕意公语所知曰吾犹嫠妇也茹荼积乆乃以白首改节耶濡迟散秩竟以三品终（见张东沙芝园集）
　　
　　黎文僖公淳公素俭朴患乡俗好侈躬自裁抑倡之婚塟饮宴之礼人多视以为则重伦尚节痛违禄养极严庙祀兄嫂卒其子及女皆幼育为己子尝归省至临清适山东按察副使董国器妻死而董使未还特使人舁其柩以归太常卿余孟亨卒家贫不能举塟公倡同乡士绅合赙襄其事乡吏邓禄寓银数十两后禄死藏所寓物候其子长还之所居黄洋渡积潦病涉捐私帑筑堤四十丈民到今利焉训诸生日亹亹不倦登甲科仕中外者相接踵下逮胥吏有向学者辄优给之后亦多成名云惟公风致若敦朴醇厚长者乃主顺天场试事时有试卷竒甚后场不类疑有弊勾稽墨卷得誊録截卷奸状按其事而取是卷为解首则名士马中锡也其精明又若此公能尽职殚心耳此弊闻时益多所从来矣李文正公东阳幼负儁才藉有清誉艺林推为神骏云路比之祥鸾其推毂天下士孜孜如不及也其为相也会逆竖刘瑾乱政毒螫缙绅公委曲周旋多所全济众颇赖之盖瑾窃太阿之柄大肆凭陵目中已无天子独公以素望稍加钦重其清约之操出自性成冬月不炉披册操觚不胜其栗辄就日而暴之日移亦移其俭如此余家尚书邦竒公门人也一日侍坐有兴化守者亦公门下士以觐事至京缄两帕四扇令从吏馈公公曰扇以染翰固可但多帕奈何吏顿首于庭乃启缄取扇而归其帕云即此一事古风可想已（见张东沙芝园集）
　　
　　公致政后邃庵杨阁老载酒肴过懐麓堂为寿觞以金公讶曰公近亦有此器耶邃庵有惭色自是不敢用以觞云
　　
　　附赵司成永号类庵京师人一日过鲁学士铎邸鲁公曰公何之司成曰忆今日为西涯先生诞辰将往寿也鲁公曰吾当与公偕公以何为贽司成曰帕二方也鲁公曰吾贽亦应如之入启笥索帕无有踌躇良乆忆里中曾馈有枯鱼令家人取之家人报已食仅存其半鲁公度家无它物即以其半载与赵公俱往公所称祝公烹鱼沽酒以饮二公欢甚即事倡和而罢
　　
　　右闻诸都人刘宪副效祖者刘耳承之赵司成云赵司成亦都人与鲁公皆公门人也
　　
　　耿子曰公仕宦五十余年柄国且十有八年矣郑端简谓公卒之日不能治丧门人故吏醵金钱赙之乃克塟又谓尝过其门萧然四壁不足当分宜辈一宴之费云云则公平生所以禔身者可知已彼时权珰狂猘公卿鲜不受其螫者而卒不敢有加于公公岂有权术牢笼之哉毋亦贞操洁履有以服其心耶
　　
　　罗文恭述杨文襄一清为相时有人馈美珠一斗者公直受之嗣诸邉将谒请留侍左右询邉事公随出所受珠分劳之投之地顷刻立尽门生有以贫归者发嚢助给率数十百金以为常盖以天下财为天下用而不为私蓄即比于一介不取可也
