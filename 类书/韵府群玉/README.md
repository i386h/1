　　钦定四库全书　　　　　子部十一
　　韵府羣玉　　　　　　　类书类
　　提要
　　【臣】等谨案韵府羣玉二十卷元隂劲撰其弟复春注黄虞稷千顷堂书目云阴劲一作阴时遇字时夫奉新人数世同居登宋寳祐九经科入元不仕其兄中夫名复春据此则时夫乃劲之字而中夫又时夫之兄与世所传不同当必有据然旧刻皆题其字未详何义也昔顔真卿编韵海镜源为以韵隶事之祖然其书不传南宋人类书至多亦罕踵其例惟吴澄支言集有张夀翁事韵撷英序称荆国东坡山谷始以用韵竒险为工盖其胸中蟠万卷书随取随有傥记诵之博不及前贤则不能免于检阅于是乎有诗韵等书然其中往往陈腐云云是押韵之书盛于宋末元初时夫是编盖即作于是时康熙中河间府知府徐可先之妇谢瑛又取其书重辑之名増删韵玉定本今书肆所刋皆瑛书此本为大徳中刋板犹时夫原书也明成祖颇喜其书故解缙大庖西封事称陛下好观韵府杂书钞辑秽芜畧无文彩曹安谰言长语亦曰韵府羣玉亦收之博矣其中正要者漏之如足字欠管宁濯足栅字欠青溪栅苏峻攻青溪栅卞壸拒之高颎杀张丽华于青溪栅二字尚失之他可知也云云今以
　　钦定佩文韵府考之阴氏之所漏宁止于是安之所举如一叶一花偶然掇拾未睹夫邓林之茂蔚也然元代押韵之书今皆不传传者以此书为最古又今韵称刘渊所并而渊书亦不传世所通行之韵亦即从此书录出是韵府诗韵皆以为大辂之椎轮将有其末必举其本此书亦曷可竟斥欤乾隆四十六年十二月恭校上
