　　説略原序
　　昔在甲午乙未之间予端居多暇案上庋説部书数十种随手取一卷讽之以代藼苏陶日月遇有可备攷质者苦其性善忘輙取赫蹏识之以类黏尺二钜册上时自哂其有掌録而无舌学已离而为二十卷名曰説略藏诸笥中中间曽一属友人半埜李君纠其错乱浙门张君厘其防画凡再易草始克成编乙巳自京师请告归舟败于南阳之决河而此书亡矣旧藁尚存张君处防其人又物故予惋惜者久之既而其家从敝笥中检以归予予颇有珠还之喜因念门类差分涉猎未广更欲厘正而附益焉而病复未果仅蕞为三十卷録而存之或谓就一类之中多有缺遗就数类之中多非闗要或或引可有可亡重言虽连犿而无伤璅事实荒唐而莫辨爱竒自可语怪何为予亦爽然自疑不能定也两弟时从旁怂慂梓于家塾以示儿軰自顾无力荏苒数年顷新安袐书吴君徳聚博古通微躭耆佚典偶过金陵访予探得数卷读而好之请事剞劂因念曩时排缵之劳与漂没之感举以付焉用成其义先是文学朱君君器见而心赏捐赀为之缮写刻既竣乃并记其所繇如此嗟乎世之闳览博物君子且囊括昔之为海为郛者以大其畜何以略为顾有如予之好臆而善忘者时探而讽之亦可借以备丹阳之钞补河东之箧虽六艺键钤九流津涉原原本本之论未极于斯至于考騐是非综校名实兼资前识用广异闻古今贤喆之用心往往可以概见是在览者择之而已若曰断截美组织纎细则予岂敢避偃鼠饮河之诮哉万厯癸丑嘉平江宁顾起元书于遯园之懒真草堂

　　日名朱明一名灵耀一名东君一名大明一名阳乌见广雅一名翔阳见木华海赋方诸山眞天呼日为圆罗耀外国人呼日为濯罗耀见眞诰羲和者帝俊之妻是生十日浴日于甘泉见山海经郁仪奔日之仙郁仪赤文与日同居见鸡跖集日中有三足乌名踆乌见汉天文志及五经通义日出旸谷浴于咸池拂于扶桑是谓晨明登于扶桑是谓胐明至于曲阿是谓朝明临于曾泉是谓早食次于桑野是谓晏食臻于衡阳是谓禺中于昆吾是谓正中靡于鸟次是谓小迁至于悲谷是谓晡时回于女纪是谓大迁经于泉隅是谓高舂顿于连石是谓下舂爰息六螭是谓悬车薄于虞泉是谓黄昏沦于蒙谷是谓定昏日入崦嵫经细栁入虞泉之地蒙谷之浦垂景在树端谓之桑榆见淮南子占日法气形直微起在日上为戴青赤气抱日上小如冠为冠青赤气小而交于日下为缨青赤气小而圆交日左右为纽青赤气如小半晕状在日上为负皆有喜青赤气长而斜倚日旁为防青赤气圆而小在日左右为珥有晕日旁如半环向日为抱背日为背青赤气如带在日四旁为璚青赤气长而立日旁为直气成形在日四方为提青赤气横在日上下为格日下有黄气三重名曰承福人主有喜且得地青白气如履在日下曰履见晋天文志
　　月中嫦娥之说学斋占毕尝论其妄云汉志黄帝使羲和占日常仪占月周官注云仪俄二字古皆音娥毛诗实惟我仪协在彼中河乐且有仪恊在彼中阿太经各遵其仪协不偏不颇以是例之仪俄二字本同音所谓常娥即常仪也予谓仪之作俄此诚可据或后人转写为娥羿妃窃药奔月之说从而傅防之耳楚词天问安得夫灵药不能固藏则其妄已久又攷之通鉴前编常仪乃帝喾之四妃是知后人谓为月娥者以有常仪占月事而又惑于帝妃之文故也
　　月有形无光日照之乃有光日似弹丸月似镜体或以为日亦似镜体见京房占月御曰望舒亦曰纎阿见淮南子月中有物婆娑者乃山河影其空处海水影亦见淮南子须弥山有阎扶树月过树影入月中见佛蔵经冬至后月养魄于广寒宫见东方朔十洲记月宫榜曰广寒清虚之府见龙城録月中有兎与蟾蜍见五经通义羿妻嫦娥窃药奔月托身于月是为蟾蜍见后汉天文志有逺飞鸡朝往夕还常啣月中桂实归于南土见洞防记月中有桂树高五百丈又月乃七寳合成常有八万二千戸修之见杂俎又考佛典言日为火珠月为水珠其广径各有由旬数皆七寳琉璃所成宫殿日天子月天子居之而道藏经又载日姓张名表字长史日中青帝名圆常元字照龙韬赤帝名丹灵峙字緑虹映白帝名皓郁将字廻金霞黑帝名澄潧渟字緑炎黄帝名夀逸阜字飊晖像月姓文名申字子光月中青帝夫人名娥隠珠字芬艳婴赤帝夫人名翳逸寥字婉延虚白帝夫人名弄素兰字郁莲花黑帝夫人名结连翘字渟属金黄帝夫人名青营襟字炅定容此等又与释典所言异且日月何以有姓氏名字纷纷傅防可为拊掌也
