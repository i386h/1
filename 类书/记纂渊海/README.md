　　钦定四库全书　　　　　子部十一
　　记纂渊海　　　　　　　类书类
　　提要
　　【臣】等谨案记纂渊海一百卷宋潘自牧撰据浙江通志自牧金华人庆元元年进士官龙防令此本题曰教授盖其著书时所居官也是书分门事与诸家畧同惟一百卷中叙天道者五卷叙地理者二十卷叙人事者六十四卷叙物类者仅十一卷详其大而畧其细与他类书小异其郡县一部以临安为首盖据南渡割裂之余而五岭两川之后更及开封诸府存东京全盛之旧亦与方舆胜览诸书删淮以北不载者体例有殊其中性行议论诸部子目未免琐碎然亦不失为赅备也此本刻于万厯己夘原本卷首自牧名后题有中宪大夫大名府知府前监察御史东鲁王嘉宾补遗字则亦如陈禹谟之改北堂书钞已非自牧之旧又陈文燧序称其先世求之闽蜀得其前编周流吴越复购后编此本不分前后编盖复经合并益失其真序文称中叶零替蠧鱼残缺戊寅冬承乏畿南公暇谬为补注剥落太甚者属别驾蔡公司理顾公学博吴君采辑诸书补阙序次一日示诸太守越峯王公邑令吴君愿捐奉梓之云云以其卷首列名考之别驾蔡公为大名府通判蔡之竒司理顾公为推官顾尔行学博呉君则有府学训导吴腾龙魏县教谕呉嶙二人不知谁指邑令呉君为南乐县知县吴定太守王公当即嘉宾是补此书者为文燧及蔡之竒等三人嘉宾特为刋板未尝操笔与题名亦互相抵牾盖明人书帕之本称校称补率随意填刻姓名不足为亦不足为异其出自谁手无庸究诘要其根柢则固宋人之旧帙耳乾隆四十六年三月恭校上

　　刻记纂渊海序
　　太史公曰学者载籍极博犹考信于六艺云夫六艺尚矣顾挽近世离合雕缋食宋氏余餲以博甲乙第则可以方韩毛戴郑则滥觞只涂学士耳目耳一旦释故筞登朝复东西南北或不暇搜百代之薮即搜而柱下所藏或不少槩见即见矣而惠子五车纷如岂胜徙代舍也则恶在其能博极哉故述家者兴其人则如刘叅军徐散骑欧阳率更其编则经史子传错其用发善机则开檐展卷古道照顔其资不可缺也博士弟子展纸濡颖镕意铸词仰天阁笔而不能成章则悔其积学之晩迨其既仕则簿牍琐冗尘纷纠纒上之树勋防防希髙慕大考镜一差不免效颦学步之过次则摸棱两可得丧交驰又次则堕行防防机械闭护其意念蒸蒸皇皇万起万防当此之时取圣贤故牍而讽诵防绎之印证先哲神逰太清投氷熄火方寸烱然与圣贤之心旷世孚格而三事九列轻若一毛即使震撼击撞排山岳而膺霆电其刚大不少馁焉则不惟定志凝神莫过于书而昔人所谓至乐者真在是矣使书而可废则臧逢世求刺尾以书班史杨司谏佣集贤以便稽阅欧阳文忠居夷陵时求汉史不得輙考吏牍以镜得失穷一生心力而不倦者岂其疲精于腐防蠧鱼之资而为是迂缓不切之事哉自昭明聚精于汉选宋帝撷秀于英华杜氏覈例于通典好古修文之士至今便之苐挈纲者不详撮要者不愽且专主采摭而无禆劝惩母怪乎博物者縁以为资而志道者谓其索然无余味也有宋金华潘自牧氏性既嗜古学亦遡源研思殚力彚集全书命曰记纂渊海自乾坤法象之理皇王帝霸之绪礼乐刑政之辨草木昆虫人物事变之纪文井蛙疑焉河伯叹之矣陈大夫由名御史出秉魏宪梓其书魏博大夫家临川临川九渊旧乡故多谭一贯者乃是书藏大夫家百年且大夫至不三月亟亟焉梓所谓考六艺而博极非耶余代大夫至则释爼豆议军旅盖十九废书一日手其编咨咨叹焉曰当世卿大夫修夏商之业具曰文胜而词赋则墨儒多阙不称即两都山川宫观礼乐兵车姬姜端冕祯祥之茂未有一赋之者谓腹笥何何令来学前无徃哲防哉编也愿捐俸梓之成分为门若干条若干卷若干盖数十年未竟之绪而焕然大备矣夫先人有志后裔弗成非孝也家有和璧秘而弗广非仁也非仁非孝非夫也自牧民采辑于数百年之久而先祖父得之则传者在先祖父先祖父访缀于十世之久而子孙得之则传者在我子孙借令惜劳省费不能公之天下谓先祖父何今是编既成烂然拱璧譬诸灵泉浩淼渴者思饮嘉生畅茂饥者可防也学士大夫溯流穷源剥华嚅实由经籍子史以归仁义道德之懿自渣滓糟粕以要精神心术之防甲乙雌黄立我正鹄释回増美身体神防处则为一世闻人出则为经天实学则经籍子史皆我注脚防古圣哲所以垂世立教者于吾人有实济而剪防醸蜜蔚为词章又卑鄙不足言矣虽然珠玑玳瑁贵矣而不可衣食布帛菽粟可衣食矣而不能必斯人之各足取不禁用不竭随其分量所及而各适于用者书也遗簮见收宝于珠玉断简复续贵于天球昔人六籍之堂万卷之架固惓惓然恐手泽之坠而未必非公世觉人之心也余祖辈不敢冺自牧公之传而余不敢冺先辈之志继绍之心得借诸公以自効而越峯诸公捐俸梓集嘉惠万世则公已公物之心亦有所寄以不朽其自所慰者因甚于予之慰其先人者也把玩是编者其尚念之哉万厯己卯孟夏吉旦赐进士第整大名等处兵备河南按察司副使前河南道监察御史临川陈文燧书

　　记纂渊海序
　　夫多识不如一贯宣圣与端木氏论之详矣彼忘书三箧了诵无遗借阅市肆过目成诵其所建树多贻讥于后世而暗合道妙者腹稿书囊且贤于读书不识字之軰则载借蘧庐圣人不屑雕虫末技岂壮夫为之哉然前言徃行大畜多识而不学无术之诮又若不病其玩物防志者是何以故也盖人生防内上自皇初下穷万禩判然若风马牛不相及而其善善恶恶泾渭不淆即万禩如出同堂刓觚裂轨之徒靡所不至夜气清明检防旦昼輙愧汗不能自遏盖天则之明不容冺耳所恃以区较臧否激则渔猎备矣于乎此率涂之人立起而渔猎者耶抑为周览经者航渊海乎且韩愈一代作者而称纪事纂言唯钩提是竞扬雄则雕虫耻之自二论狎出搜罗驰骋之士与着一家言言各屈信龙蛇若廼缙绅先生更抗手而尊一贯曰赐氏固病博矣然则夫子焉学而识防风氏之骨肃慎氏之楛矢防龙罔象羵羊之秘恠若河汉无极耶且窥孔门者即博士家言童习而斑白不敢废何独废学欲黜类苑以下书籍令刘徐其人一生当游夏无论文学其多识鸟兽草木之名亦何渠尽见黜也语有之善识者譬善货殖谷纑旄石漆丝姜桂柟梓瑇瑁齿革连锡旃裘之类靡不韫匵岂必鞶鉴舒鼎而藏垂棘而寳哉故宋潘子犹然俭诸氏而饶益之世母必齐梁晋魏秦汉春秋事母必庙彛国典言母必邹枚迁固屈宋左国老荘盈耳者张夺目者陈书成而命之曰纪纂渊海有味乎其渊海也而经籍子史诗赋风骚变而外国盗贼水火灾沴防而医卜农圃稗官小説莫不具陈而其因事别类因人证事治政之纯疵人品之髙下行谊之淑慝心术之忠邪华衮斧钺凛凛可畏盖兼经史之义而集诸书之大成者也余先世强恕公牧石诸公求之闽蜀得其前编周流吴越复购后编寳玩盖几百年所矣中叶有志续蠧鱼残缺余自通籍时业有志续之践更南北竟无完书今戊寅冬承乏畿南公暇谬为补注剥落太甚者属别驾蔡公司理顾公学吴君采辑诸书补缺序次一日示诸太守越峯王公邑令吴君则皆唯唯曰后启班固左思嚣嚣上世也孰为我封魏卢柟之墓讨其乡岂复有赋沧海者乎且诏之曰陈大夫书出丰玉荒谷积恣而所贸布矣太守滕阳王君梓既竣入谓不佞故不佞稍扬扢云尔诸诠次卷帙不载载大夫自序中万厯己夘中秋日赐进士第中宪大夫整饬大名等处兵备河南按察司副使前奉提督云南学校巡按直福建江西道监察御史勾余胡维新书

　　太极【混沌附】
　　经易有太极是生两仪疏云老子道生一此太极是也【易】夫礼必本于太一【记】
　　子有物混成先天地生【老】太易者未见气也太初者气之始也太始者形之始也太素者质之始也气形质具而未相离故曰浑沦　牝之门是谓天地之根【并列】未有天地自古以固存神鬼神帝生天生地在太极之先而不为髙在六极之下而不为深【荘子】太初有无无有无名一之所起有一而未形　儵与忽时相与遇于混沌之地混沌待之甚善儵与忽谋报混沌之徳曰人皆有七窍以视聴食息此独无有尝试凿之日凿一窍七日而混沌死【并荘】天地未形冯冯翼翼洞洞故曰大昭天地未剖隂阳未判四时未分万象未生汪然平静
　　寂然澄清莫见其形　古未有天地之时唯象无形窈窈芒芠漠闵澒蒙鸿洞莫知其门【淮南】太极初搆清浊始分【抱朴】夫子其与太极合徳神道并行乎【文中】
　　史太极元气函三为一【前律厯志】太极中央元气故为黄钟太极运三辰五星于上　经元一以统始易太极之
　　首也【并同上】太极者混沌之时曚昧未分日月含其辉八卦隠其神天地混其始圣人藏其真【晋纪瞻传】
　　传记太极形质已具【帝王世纪】气形之初谓之太始【同上】先有太初后有太始形兆既成名曰太素浑浑相连视之不见聴之不闻【白虎通】
　　集邃古之初孰传道之上下未形何由考之【楚词天问】太易之始太初之先冯冯沉沉奋搏无端【扬雄赋】太极之先两仪始分有沈而奥有浮而清【班固典引】览太极之初化【陆机赋】嘉太极之开元【傅赋】太极剖判造化权舆【左思赋】太极之始混沌未分【曹子建七啓】在昔太初黄混并混沌鸿蒙兆眹未形【曹子建论】易谓之太极春秋谓之元老子谓之道【阮籍论】无极之极漭弥非垠【柳文天对】本始之茫诞者传焉鸿灵幽纷曷可言焉【同上】混沌无形气奚从生两仪元一是能分太极焉得离【张萃诗】
