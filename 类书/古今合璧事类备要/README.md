　　钦定四库全书　　　　　子部十一
　　古今合璧事类备要　　　类书类
　　提要
　　【臣】等谨案古今合璧事类备要前集六十九卷后集八十一卷续集五十六卷别集九十四卷外集六十六卷宋谢维新编维新字去咎建安人其始末未详自署曰胶庠进士盖太学生也是书成于寳祐丁巳前有维新自序后有莆田守黄叔度防称维新应友刘徳亨之托盖当时坊本总目后又有防云昨刻古今备要四集盛行于世但门目未备再刻外集云云不署名氏当即徳亨所题也是书前集四十一门子目四百九十一后集四十八门子目四百一十六其致仕一目有录无书注曰已见前集续集六门子目五百七十别集六门子目四百一十外集十六门子目四百三十所引最为详悉惟郡县山川名胜以祝穆方舆胜览已备不及更载焉每目前为事类后为诗集所收皆兼及宋代皆不及太平御览册府元龟诸书皆根柢古籍原原本本而所采究皆宋以前书多今日所未见宋代遗事佚诗如苏轼咏雪诗以富贵势力分四首为本集所不录者亦徃徃见于此书故厉鹗作宋诗纪事多采用之又宋代官制至为冗襍宋史不过仅存其名当时诗文所称今多有不知为何官者惟此书后集条列最明尤可以资考证在类事之家尚为有所取材者矣乾隆四十六年五月恭校上

　　古今合璧事类备要原序
　　类书之编何所昉乎自尔雅载虫鱼之名陆氏疏草木之辨沿而下之至于孔白六帖出而类书备矣虽然犹未备也两坊书市以类书名者尚矣曰事物纪原曰艺文类聚最后则锦绣万花谷事文类聚出焉何汗牛充栋之多也今友人刘兄以类书见嘱且以合璧事类备要名岂非欲备所未备又摭其要而为之备乎其视夫徒载首尾而不得其要□分门目而备失其伦者万万不侔昔人有不善记事每求一事为之防绎搜索至有终日追思而莫穷其所出者今而是编始而天文地理次而节序人物以至族属官职姓氏之分儒学仕进道释技艺之等与夫吉凶庆吊冠婚丧祭之仪草木虫鱼器用动什之末莫不类而得其备备而得其要其间别以标题配以合璧俾阅是编者求其一则知其二观于此则得于彼既无摉摘之劳而有骈俪之巧如逰元圃而取瑶宝入武库而缮甲兵则其有功于后之类书者多矣言未毕客有诮者曰果止于备要何其方帙如是之浩瀚耶余应之曰今之所编正将备前乎未见之书以充后乎无涯之用凡古今应用之事悉于此书萃焉又何以讶其多为客曰诚如是盍以合璧备要名余曰唯遂绣诸梓宝祐丁巳大吕既望胶庠进士建安谢维新去咎父序

　　古今合璧事类备要跋
　　记问之学不足以为人师然左氏有广记备言之要乃为后世类书者之祖一旦友生谢去咎携一编以示余曰此友人刘兄德亨所托编事类备要也不私所藏欲广其传且俾余一言以订其末余不敏何所容喙傥休沐之暇时阅此编信足为挥毫用事之助云郡人新莆田守黄叔度似道敬跋

　　事类羣物之祖【天者丨丨丨丨也故徧覆包含而无所殊建日月风雨以和之经隂阳寒暑以成之前汉董仲舒传】羣阳之精【天丨丨精也合为太一分为殊名故立字一大为天春秋説题】高目下耳【敢问天聪明曰昭昭乎惟天为聪惟天为明夫能丨其丨而丨其丨者匪天也夫扬问明】积气成形【杞国有忧天地崩坠者晓之曰天积气耳亡处亡形奈何忧其崩列子】道犹张弓【天之丨其丨丨丨乎高者抑之下者举之有余者损之不足者补之老子】形如倚葢【周髀家云天丨南高而北下日出高故见日入下故不见天之居丨丨丨故极在人北是其证也极在天之中而今在人北所以知天之丨丨丨丨晋天文志】行如蚁磨【天员如倚葢地方如棊局天旁转半在地上半在地下日月本东丨天西旋入于海牵之以西丨丨行丨上磨左旋蚁右行磨疾蚁迟不得不西同上】运如车毂【天形如鸡子天大地小表里有水地乗气而立载水而浮天丨丨丨丨之运浑天仪】天有四时【春秋冬夏风雨霜露无非教也孔子间居】天有六气【隂阳风雨晦防也左】无为而成【哀公曰敢问何贵乎天道也子曰贵其不已如日月东西相从而不已是天道也不闭其久是天道也丨丨丨丨是天道也已成而明是天道也哀公问】不言而信【天聪明也圣人得之天垂象也圣人则之丨丨丨丨不怒而威验之以衡轴考之以枢轴六帖】四时天名【春为苍天夏为昊天秋为旻天冬为上天于春言色于夏言气于秋言情于冬言位相备也尔雅】九方天名【九天东方皥天东南方阳天南方赤天西南方朱天西方成天西北方幽天北方天东北方变天中央钧天楚辞注】五方天神【天神之大者曰昊天上帝即耀魄宝也亦曰天皇大帝亦曰太一其佐曰五帝东方青帝威灵仰南方赤帝赤熛怒西方白帝白招拒北方黒帝汁光纪中央黄帝含枢纽云出五经通义】六天欲界【黄防天玉完天何童天平育天文举天摩夷天凡丨丨为丨丨也防命经注】十二天界【越衡天蒙翳天和阳天恭华天宗飘天皇笳天堂曜天端静天恭庆天极瑶天元载天孔升天皇崖天合丨丨丨为色丨同上】十四天界【极风天孝芒天翁重天江由天阮乐天昙誓天霄度天元洞天妙成天禁上天常融天玉隆天梵度天贾奕天合是丨丨丨为无色丨同上】周天行度【纪星二十八宿布于四方东方苍龙七宿其属三十二星七十五度北方武七宿其属三十五星九十八度四分度之一西方白虎七宿其属五十一星八十度南方朱雀七宿六十四星一百一十二度丨丨三百六十五度四分度之一每一度二千九百三十二里周天积一百七万九百一十三里径三十五万六千九百七十一里后地理志注】天地相去【晋书天文志曰日去地常八万里又曰地上去丨八万一千三百余里又地理志丨东西南北丨丨二亿三万余里自地至天半八极之数酉阳杂爼云自地至天十一万余里学林】天在西方【呉使张温来聘温曰丨有头乎在何方秦宓川人曰有丨丨丨诗曰乃眷西顾以此推之温曰天有耳乎宓曰天处高听卑诗曰鹤鸣于九臯声闻于天无耳何以听之温曰天有足乎宓曰天歩艰难无足何以歩之温曰天有姓乎宓曰姓刘天子姓刘以此知之蜀志】号玉皇君【木公亦曰东王公葢青阳之气万神之先也亦丨丨丨丨居于云房紫云为葢青云为城直僚仙官皆禀其命汉有小儿歌曰着青裙入天门揖金母拜　公乃木公之玉童也太平广记】放状元榜【范仲淹倅陈州时郡守召道士奏章五更乃起守问今夕奏章何其久也曰方出天门遇丨明年进士榜观者骈道以故稽留公问丨丨何姓曰姓王二名下一字黑涂旁注一字远不可辨及明春状元乃王拱夀御笔改为拱辰括异志】梦翼登天【陶侃丨生八丨飞而上天见天门九重比丨其八唯一门不得入阍者以杖击之因坠地折其左翼及寤左腋犹痛倦游録】以手捧天【韩魏公知防州卧疾数日忽曰适梦丨丨丨丨者再其后援英宗于藩邸翼神宗于东宫同上】链五色石【昔者女娲氏丨丨丨丨以补天阙断鳌足以立四极列子】记白玉楼【李贺梦人持版告之曰天帝新白玉楼请君为记贺遂卒李贺传】钧天广乐【秦穆公梦至帝所观丨丨丨丨帝锡之以防秦遂大昌史书】红云帝居【玉帝所居常有丨丨拥之虽眞仙亦不得见其面翼圣传】天门千骑【君之来兮丨丨空从丨丨兮驾飞龙云云坡上清辞】玉台羣仙【君之去兮天门开欵阊阖兮朝丨丨丨丨迎兮塞云汉俨前导兮纷后陪同上】虎豹九关【丨丨丨丨兮啄害下人些楚辞】天皇琼宫【呌帝阍使辟扉兮觌丨丨于丨丨出张衡赋】
