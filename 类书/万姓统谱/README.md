　　万姓统谱自序
　　余读眉山苏氏族谱引感而辑姓谱云引之言曰凢人情见乎亲亲见乎服至无服而遂途人焉其初兄弟也一人分焉者也又曰一人分而至途人势吾无如之何也先其势而图之使无忽忘焉可也夫即其言而绎之将以联苏氏一家之心已也然握其机以联之者庸他术乎仅惟族谱之一书故苏氏自谓观吾谱者油然而生孝弟之心焉夫天下家积也谱可聨家矣则聨天下为一家者盍以天下之姓谱之夫自开辟来遐哉邈乎生齿繁育总总林林受姓而载简册者不啻百焉千焉而殆万也夫岂生于空桑乎又岂各为门户乎上古虚静恬淡寂寞无为民熈熈相忘于世如鱼相忘于江湖岂岐人已而二之逮徳下衰淳散朴离日以防鸢蜗角所争者而奔走焉至分胡越于一膜之外遡其所自岂不以面目本来判焉若风马牛不相及乎而岂知万榦一本万一源也考之世谱曰五帝三王无非出于黄帝之后黄帝一十五子而得姓者十四徳同者姓同徳异者姓异则知凡有生者皆一人之身所分也分而以嗣以续愈逺愈繁由一人而百姓而千姓而万姓虽棼焉杂焉散漫而不可统括实一本一源之枝流耳而相倾相轧胥虐胥戕施之胡越且不可矧于一膜之外耶知此则聨天下为一家反掌耳故观吾之姓谱者孝弟之心或亦可以油然而生矣此余辑谱意也噫天地大父母也凡物皆受于天地而始为象【阙】则凡物且皆吾与必咸若之而后【阙】　周流贯彻况同此形性者耶而况同此受姓者耶古之君子有不敢【阙】附于逺祖之显者昩也有必欲自【阙】而涕泣者隘也其于大本大源何【阙】谱成次一百五十巻条列载之凡例兹不复赘识者慎毌曰有是哉子之迂而哂也吴兴凌迪知譔

　　万姓统谱自序
　　凡例
　　一姓氏一书旧不下数十种有论地望者有论国氏者有论声者有论字者有仿姓书编者夫论地望如世本王侯大夫谱姓氏英贤録是也乃以贵贱为主然贵贱无常安得专主地望论国氏者如氏族要状通志氏族是也乃以本源受氏为主然乏世系者复列以韵则混淆无辨徒乱耳目论声者乃以四声为主如姓氏韵畧姓源珠玑是也然平仄不调东冬不别以梁惠王为梁以齐宣王为齐则舛缪可鄙何取于姓也论字者乃以偏傍为主如仙源类谱姓氏秘畧是也然拘于防画不论其理但可为字书于姓氏无与也有仿姓书编者如合璧事类尚古类氏翰墨全书是也然数止四百二十二家族系未广而千姓编一巻又工于组织搜罗未备且诸书皆止述先朝未及昭代非为全书也是编逺自上古近迄昭代凢系姓氏罔不萃聚以四声韵为纲以东冬支微为目更考韵玉韵防等书凡韵字下注姓者以次备载庶几海内古今之姓悉具简册而名贤硕士一举目无遗矣
　　一姓有望有音有氏曰望者如河间渤海等郡名是也曰音者宫商角征羽是也曰氏者如以国以邑以官为氏等类是也是编以韵为先后以朝代为次序每一姓下一曰望二曰音三曰氏使姓各得所系之本不失受氏之宗庶为氏族全书也
　　一希姓无人者惟书其姓并望氏或仅有其姓而无人无望氏者必出于韵书姓苑姓纂原载始録之然或韵书姓苑姓纂所不载而更有数人森列者古则凖于成书今则凖于志録若无所据不敢强书以紊世系
　　一姓之有氏如水之有源歴代世谱云五帝之相承三王之继禅无非出于黄帝之后故有姓必有氏今常姓亦有无氏者而希姓则漫无考据竟不得其所系之本兹惟仍旧注缺以存其姓
　　一姓之有望以其著名于郡故曰望本源则同而支派各异如刘则二十五望王则二十一望余或五望四望二三望者然惟取其显者书之间亦有无望者悉从旧借不敢妄加增损
　　一希姓间有韵府诸书无字者考海篇直音附入如直音字有注姓者原系某字今防画更易作某字者亦附注于末以存其姓
　　一韵书以沈约四声为主第中间有一字而平上去入俱有者惟据原韵注下有姓字者得入如一东之逄二冬之逢五微之贲四寘之贲九蒸之能七泰之能是也若韵下不注姓字而姓纂姓苑所收者亦并録入
　　一复姓旧书俱错杂无叙兹取上字为据亦分四声别为六巻
　　一关西代北二字四字诸姓自为一类不附于四声之末至于辽金元三朝臣士如托克托赛音布哈超尔诸人着于史册各有氏族可考然音译或未画一仍俟订正
　　一姓谱之书虽以纪姓然因姓而考其人必言可为法行可为则声施当时垂名后世者始可入録或一节一行之善亦所不遗若奸恶昭彰人所共知者则删之不敢有污简册至如希姓或名微而官存或事微而无善可纪亦并纂入即行有颇僻者间亦书之以存姓也稍为分注以别善类
　　一人物歴履采于二十一史列传及通志统志郡邑志等书若近代未登志者稽各文集并志状以州里勲阶为主其余言行事迹惟诠述梗槩而已或无州里而祗具一事或有州里而无事可述或州里与事俱缺但存其名者悉徴故牒载之不敢创一言也
　　一常姓如张王李赵之类颇繁剧不能槩书惟取其事之昭彰人之显著方始编入余则在所畧也
　　一姓书旧有道释闺婉二类然道流衲子或称号或称居或称法名或称字往往不注姓字而诸家姓氏未及精研遂以贯休和尚作贯打地和尚作打浮邱伯为浮真子为真是乱姓统矣闺婉女流又多以夫姓为主又不可不辨也故并删之
　　一前代人物昭于典借者无论小贤大贤备録不遗若国朝人物藏之金匮石室而野史所载者未详耳所习闻惟鸿卿钜公遭时遇主论思宣力临难抗言仗节死义以至文学论议定是稽疑与闾巷布衣砥行立名者悉登载之若曲谨小亷非济济多士之朝所尚也不敢槩入至于希姓虽丞尉小官亦得并书不在是例
　　一本朝先贤并近时新谢世名公间亦仅具爵里不备载其徳行功业者限于未得其志状之实既不敢遗而不书又不敢妄加评品故书之以俟他日
　　一本朝人物常姓惟已谢世者生则俟之盖不愸之老尚未定于盖棺故不敢入至若希姓虽见列仕借者不遗但分注以示别尔
　　一本朝人物细注凡遇帝上防御等字宜擡头者则虚一字不复更提从古今石文例也
　　一春秋战国人物希姓者具在左传諌诤论议全文未可以章句割裂故仅书其事之因或具其姓名而已不及详载
　　一每韵内常姓字在前希姓字在后若拘音韵之次第排序则烦简混淆不便观览
　　一世代人物或有字者无字者或有地者无地者或有官者无官者悉从旧籍不必以定例拘之也
　　一是编引用诸书甚繁大畧以姓氏等书为宗参用一统志十三省通志各郡县志至于经史子集等书凡有一人一姓可録者悉采择类编然恒以挂一漏万是惧如有耳目未习遗于载籍之外或知识未广误改金根者幸博古君子正之
