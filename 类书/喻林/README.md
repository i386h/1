　　钦定四库全书　　　　　子部十一
　　喻林　　　　　　　　　类书类
　　提要
　　【臣】等谨案喻林一百二十巻明徐元太撰元太字汝贤宣城人嘉靖乙丑进士官至刑部尚书是书以经史及诸子集中所载古人设喻之词广徴博采彚为一编分立十门毎门又各分子目凡五百八十余类歴二十余年而成复自加刋削而付之梓用心颇为勤至其引书各于条下注明出处篇目次第胪载无遗亦逈异明人剽窃撦挦之陋习特其自序称阅书四百余种而检核书中所列篇籍实不及其半未免涉于张又往往随手摭拾不能得其根源如儿説宋人善辨者一条本出韩非子周人有仕不遇者一条本出王充论衡而皆引作艺文类聚又懐金至者至不生归一条本出后汉书耿弇而引作文选李善注又头白可期汗青无日一条本出唐书刘知几而引作事文类聚又天寒即飞鸟走兽尚知相依一条本出沈约所作阮籍咏懐诗注而以为李善此类甚多皆为失于疎畧葢其学惟知以泛滥自矜奥博而不能考核精详故毎蹈舛漏之病然自秦汉以迄六朝文人词赋多以罕譬为工恣肆汪洋大都得力于此元太是编搜罗极备零玑断璧均足为缀文者沾丐之资犹为有禆艺苑以视坊本类书之丛杂无章者固胜之逺矣乾隆四十四年正月恭校上

　　喻林自叙
　　予至陋且善忘虽稔习篇章乆则惘然莫忆故日随防记以识所闻然尤嗜喻言为假譬之殊形异类甚易窍穿壅遏脩词者莫可合旃耳苐唐而后学士大夫骚人墨客咸嗔繁秽乃断六朝以上自经史子集以及道佛诸书四百余种凡语渉比辞者无论圣贒与流略之粕华目所尝见必手録焉即非喻而可为喻犹笔存之颇慙不急是勤宁免诮生玩丧惟欲砭遗忘之疾且借鲜逸心信无补空踈知犹贒乎已已遡从释褐计有二纪退食罔虚积成钜帙又病樷穰蔓杂不便省观因加诠次成编析分十门列类五百八十有竒摠彚乙百二十卷廼若粃糠沙石槩未淘亦丝蒯竝收期无漏弃也任江河之多胔自不妨汲祭其中或择而弗精将渍白蝇于清醑矣以林名喻岂尽梧哉用当厥材丑犹化好脱在唲齵之郄卒然防揳有资则匠人碎用足供庻兹辑亦非累瓦者此皆督抚巴蜀三征西南之暇所就防中州何侍御公代天来狩相与提衡为契洽偶见抄本谓宜宣布防中以惠后学予弗获辞竟灾梨要之拾唾为文人莫我知亦莫我罪比学宪郭君愽通今古一代文人海以内仰之不啻麟角凤毛而序云彚喻为林自予始予初心岂作始哉至云与予谈天下事若得于喻喻者深私心窃有厚媿夫喻林不足稐文即缪以文许则文之与武两途奚翅迳庭三征固恭仗天威而受益郭君良非渺浅嗣郭君秉钺黔中拓土开疆元老壮猷震乎殊俗诚文武为宪质之吉甫宁过逊欤予病卧敬亭间复翻阅稍为增删一二犹子胥庆雅有同嗜取而订其鲁鱼帝虎因侈口兹辑匪直骚坛之楩楠抑亦萟苑之栴檀也当与御览等编争竒芳于邓林遂镌之剞劂三征寔録后藏诸家塾庸广厥传予叙其始末如此

　　喻林卷一　　　　　　　明　徐元太　撰造化门【一】
　　天道
　　鼓之以雷霆润之以风雨日月运行一寒一暑【易系辞上传】雷以动之风以散之雨以润之日以晅之【易说卦传】
　　万物尊天而贵风雨所以尊天者为其莫不受命焉也所以贵风雨者为其莫不待风而动待雨而濡也若使万物释天而更有所受命释风而更有所仰动释雨而更有所仰濡则无为尊天而贵风雨矣【管子版法解】
　　天者诚其日德也日诚出诚入南北有极故莫弗以为法则天者信其月刑也月信死信生终则有始故莫弗以为政天者明星其稽也星不乱各以序行故小大莫弗以章天者因时其则也四时当名代而不干故莫弗以为必然天者一法其同也前后左右古今自如故莫弗以为常【鹖冠子王鈇】
　　天之处高其听卑其目芒其视察【贾子耳痹】
　　四时者天之吏也日月者天之使也星辰者天之期也虹霓彗星者天之忌也【淮南子天文训】
　　天道者无私就也无私去也能者有余拙者不足顺之者利逆之者凶譬如隋侯之珠和氏之璧得之者富失之者贫【淮南子览冥训】
　　夫至明者其照无疆至晦者其暗无疆【春秋繁露奉本】
　　夫本末相兼犹手背之相包一身和则百节皆适天道顺则本末俱畅【郭子注庄天地】
　　春秋汉合孳曰强干弱流天之道【李善注文选两都赋】
　　使星司夜月司时犹使鸡司晨也【艺文类聚天部月】
　　斗极天之中也言天者必拟之人故自脐以下人之阴也自极以北天之隂也【太平御览天部下】
　　日月若悬璧五星若编珠【初学记天部星注】
　　天有四时五行日月相推寒暑迭代其转运也和而为雨怒而为风散而为露乱而为雾凝而为霜雪立为蚿此天地之常数也若四时失运寒暑乖违则五纬盈缩星辰错行日月薄蚀彗孛流飞此天地之色诊也【法苑珠林卷六十三】
　　形气
　　天形穹隆如鸡子幕其际周接四海之表浮于元气之上譬如覆奁以抑水而不没者气充其中故也【晋书天文上】虹霓也云雾也风雨也四时也此积气之成乎天者也山岳也河海也金石也水火也此积形之成乎地者也知积气也知积块也奚谓不坏夫天地空中之一细物有中之最巨者【子天瑞】
　　清阳者薄靡而为天重浊者凝滞而为地清妙之合专易重浊之凝竭难故天先成而地后定【淮南子天文训】
　　天地之间有阴阳之气常渐人者若水常渐鱼也所以异于水者可见与不可见耳其澹澹一也然则人之居天地之间其犹鱼之离水一也【春秋繁露如天之为】
　　山犹人之有骨节也水犹人之有血脉也【王充论衡祀义篇】地以名山为辅佐石为之骨川为之脉草木为之毛土为之肉三尺以上为粪三尺以下为地【博物志】
　　夫天者统元气焉非止荡荡苍苍之谓也地者统元形焉非止山川丘陵之谓也人者统元识焉非止圆首方足之谓也【文中子立命】
　　宋均曰浑浑混混鸡卵未分也【李善注文选江赋】
　　元气周行终始无已如水之流不得独处【李善注文选幽通赋】众星累累如连贝江河四海如衣带【李善注文选陆士衡赠顾公眞诗】地常动不止而人不知譬如闭舟而行不觉舟之运也【李善注文选张茂先励志诗】
　　日薄星廻穹天所以纪物山盈川冲后土所以播气【文选陆士衡演连珠】
　　夫地有形而天无体譬如火焉烟在上灰在下也【太平御览天部下】
　　造化权舆以为东方之气风也故冻非东风不能解湛非东风不能溢【埤雅天道门风】
　　风经曰调畅祥和天之喜气也折扬奔厉天之怒气也【埤雅天道门风】
　　仰观天形如车盖众星累累如连贝【初学记天部星注】
　　土精为石石气之核也气之生石犹人筋络之生爪牙也【初学记地部石】
　　日处君而盈度月象臣而亏云逶迤而上布星离离以舒灮【初学记文部文字】
　　夫土地皆有形名而人莫察焉有龟龙体有鳞凤貌有弓弩势有斗升象有张舒形有塞闭容有隐眞之安有累卵之危有膏腴之利有塉埆之害【事类赋地部地注】
　　地东西为纬南北为经山为积德川为积刑高者为生下者为死丘陵为牡川谷为牝【事类赋地部地注】
　　天不足西北地不足东南西北为天门东南为地戸天门无上地戸无下【玉海天文天文书上】
　　昔者周公问于商高曰夫天不可阶而升地不可得尺寸而度请问数安从出商高曰数之法出于圆方【玉海天文天文书上】
　　岐伯曰言天者求之本言地者求之位言人者求之气交帝曰何谓气交岐伯曰上下之位气交之中人之居也故曰天枢之上天气主之天枢之下地气主之气交之分人气从之万物由之此之谓也【黄帝素问内经六微旨大论】易称天盖取幽防之名庄说苍天近在远望之色于是野人信明谓旻青如碧儒士据典谓干黑如漆青黑诚异乖体是同儒野虽殊不知是一【法苑珠林卷二】
　　流行
　　阖戸谓之坤辟戸谓之干一阖一辟谓之变往来不穷谓之通【易系辞上传】
　　鸟兽俄呦呦俄旬旬俄逃逃草木俄茁茁俄停停俄萧萧天地不能畱圣人不能系有运者存焉尔【闗尹子八筹】鬻熊曰运转亡已天地宻移畴觉之哉【列子天瑞】
　　杂乎芒芴之间变而有气气变而有形形变而有生今又变而之死是相与为春秋冬夏四时行也【庄子至乐】钧旋毂转周而复匝【淮南子原道训】
　　日出旸谷入于虞渊莫知其动须臾之间俛人之颈【淮南子说林训】
　　阳阴之时虽异二体常存犹如一鼎之水而未加火纯阴也加火极热纯阳也纯阳则无阴气息火水寒则更阴矣纯阴则无阳气加火水热则更阳矣【春秋繁露雨雹对】三统者天地人之始道之大纲也五行者品物之宗也道以三兴德以五成故三皇五帝三王五伯至道不远三五复反譬右循连镮顺鼎耳穷则反本终则复始也【风俗通五伯】
　　雷风焕与物时行【太经图】
　　日有南有北月有徃有来日不南不北则无冬无夏月不往不来则望晦不成【太经太告】
　　夫地之有百川也犹人之有血脉也血脉流行泛扬动静自有节度百川亦然其朝夕往来犹人之呼吸气出入也【王充论衡书虚篇】
　　星宿之相次如岁月之相袭新故相易周而复始【太经第六养解】
　　夫五行四时佐天地造物者也然水火相践金木相代而共成陶钧之致春秋异寒暑继节而俱济一岁之功【李善注文选陆士衡演连珠】
　　剖而为二判而为五或为白黑或为水火或为酸咸或为征羽【道德经指归卷八】
　　生克
　　露虽湛湛然见阳则干【毛诗湛露注】
　　五行四时十二月还相为本也五声六律十二管还相为宫也五味六和十二食还相为质也五色六章十二衣还相为质也【礼记礼运】
　　譬如火斜照水日斜照镜则水镜之灮防照他物若使镜正当日水正当火则水镜之灮不能有照日之夺月亦犹是也【春秋正义桓公三年】
　　惟火之为物能镕金而销之能燔木而烧之【关尹子四符篇】链土生木链木生火链火生云链云生水链水反土链甘生酸链酸生辛链辛生苦链苦生咸链咸反甘变宫生征变征生商变商生羽变羽生角变角生宫【淮南子地形训】金胜木者非以一刃残林也土胜水者非以一墣塞江也【淮南子说林训】
　　夫大生小多生少天之道也故丘阜不能生云雨涔水不能生鱼鼈者小也牛马之气蒸生虮虱虮虱之气蒸不能生牛马故化生于外非生于内也【淮南子泰族训】
　　凡天地之物乘以其泰而生厌于其胜而死四时之变是也故冬之水气东加于春而木生乘其泰也春之生西至金而死厌于胜也生于木者至金而死生于金者至火而死春之所生而不得过秋秋之所生不得过夏天之数也【春秋繁露天地之行】
　　金木交刑水火相敌休废于时吉凶生焉【京氏易传随上】阴阳死于位生于时死于时生于位进退不可诘正盛则衰来正衰则盛来【京氏易传涣中】
　　刚柔相伤火烂销金【焦氏易林大壮之遯】
