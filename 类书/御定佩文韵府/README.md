　　钦定四库全书
　　御制佩文韵府序
　　朕万几在御日昃宵分未遑自逸时当燕闲不辍问学羣经子史诵其文而晰其义矣以至百家之书凡可以禆世敎励民风者脩眀补正防使阙遗尝谓韵府羣玉五车韵瑞诸书事系于字字统于韵稽古者近而取之约而能博是书之作诚不为无所见也然其为书简而不详略而不偹且引据多误朕毎致意焉欲博稽众籍着为全书爰于康熈四十三年夏六月朕与内直翰林诸臣亲加考订证其讹舛增其脱漏或有某经某史所载某字某事未脩者朕复时时面谕一一增録渐次成帙犹以故实或未极博于十月复命阁部大臣更加搜采以裒益之既有原本增本又有内増外増将付剞劂矣名曰佩文韵府随于十二月开局武英殿集翰林诸臣合并详勘逐日进覧旋授梓人于五十年十月全书告成共一百零六卷一万八千余页嚢括古今网罗钜
　　细韵学之盛未有过于此书者也书成诸臣请序朕念自初至今经八年矣歴寒暑之乆积嵗月之勤朕于此书政事之暇未尝惜一日之劳也朕又尝谕诸臣从来着一大书非数十年之功不能成今数年以来所成大部书凡十有余种若非合众人之力岂能刻期告竣故念先后预事诸臣皆命列名其中兹序佩文韵府因备记编撰之始末遂及脩集诸书之大指以见成书之不易如此

　　提要
　　【臣】等谨案
　　御定佩文韵府四百四十四卷康熙五十年圣祖仁皇帝御定考唐书艺文志载顔真卿韵海镜原二百卷释皎然陪顔使君修韵海毕东溪泛舟饯诸文士诗有外史刋新韵中郎定古文菁华兼百氏缣雅备三坟句其自注又有鲁公著书依切韵起东字脚语然则分韵事始自真卿今其书不传宋元间作者颇伙谓之诗韵【语详韵府羣玉条下】其传于今者惟韵府羣玉为最古至明又有五车韵瑞然皆疎漏不完舛讹相踵杨慎作韵藻朱彞尊作韵粹其子昆田又作三体摭遗皆欲补阴氏凌氏之阙而仍未赅备是以我
　　圣祖仁皇帝特诏儒臣搜罗典籍辑为是编毎字皆先标音训所之事凡阴氏凌氏书所已采者谓之韵藻列于前两家所未采者别标增字列于后皆以两字三字四字相从而又各以经史子集为次其一语而诸书互见者则先引最初之书而其余以次注于下又别以事对摘句附于其末原本不标卷第但依韵厘为一百六卷而中分子卷二十有四今以篇页繁重编为四百四十四卷自有韵府以来无更浩博于是者俯视隂氏凌氏之书如沧海之于蠡勺矣考康熙五十九年大学士王掞等恭制韵府拾遗序有曰佩文韵府书成卷帙一百有六闻诸臣分纂之时毎缮初稿先呈
　　御览我
