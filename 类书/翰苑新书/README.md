　　钦定四库全书　　　　子部十一
　　翰苑新书　　　　　　　类书类
　　提要
　　【臣】等谨案翰苑新书前集七十卷后集上二十六卷后集下六卷别集十二卷续集四十二卷原本不着撰人名氏防明陈文烛序亦但穪为宋人今别有刋本题宋谢枋得撰者坊贾所赝托也文烛序称是书旧无本分宜袁炜为大学士时始从内阁録出而日乆佚其首卷后得华亭徐阶所録本乃足成之其书分前后别续四集疑未必出一人之手前集皆为书啓之用自一卷至六十卷皆以职官分目下至盐官酒官之类亦皆备载六十一卷至七十卷则以家世阀阅座生门生之类分目毎门之中皆冠以厯代事实次以宋朝事实次以自叙次以旁引次以羣书精语次以前贤诗词次以四六警语后集上备表牋之用一卷至十九卷以大典礼分目而附以谢恩陈乞二十卷至二十六卷则録宋代表笺之文后集下一卷至五卷为类姓六卷则惟列发举词科入学诸目葢补前集之遗别集皆録宋人劄状致语朱表表文青词疏语册文祝文祭文之属其劄子以五提头七提头九提头分目盖当时之式其朱表则青词类也续集録宋人书啓一卷至二十三卷以官分目二十四卷至四十二卷以事分目又以广别集未备之体耳其书本为应酬而作惟取便检用不免伤于烦复而于宋代典故事实最为赅备披沙防金徃徃见寳较孔续六帖之类反为有资考证也乾隆四十四年九月恭校上

　　翰苑新书前集卷一
　　三公
　　縂三公【太师太傅太保古官殷太甲时伊尹为太傅纣时胥余为太师汉仪注成王作周官立太师太傅太保兹惟三公尚书汉承秦制阙三师之位吕后以王陵为太傅陵免官省平帝元始年始置太师太傅太保太师位在太傅上太保次太傅汉百官表太师位在三公上崇号为上公东汉以后常曰上公初学记其后或置或废大抵无所统职至后魏特称三师以正其名正一品然非道徳崇重不居其位无其人则阙之六典后周改为三公隋氏又为三师李唐因之国朝之制以太师太傅太保为三师职畧政和二年诏曰太师太傅太保此古三公之官为宰相之任今为三师然古无三师之称合依三代为三公官不必备惟其人为宰相之任若除三公即为宰相靖康末诏依元丰故事以仆射为宰相三公止为阶官不复与三省之政焉职畧国朝三公官未始兼备惟元丰末文潞公守太尉雍王曹王守司空富郑公曹济阳守司徒皆同一时其后宣和间蔡鲁公为太师王将眀为太傅郑达夫为太保方相继两见石林燕语三公自祖宗以来罕尝官备独宣和末三公至十八人三少不论也太师三人蔡京童贯郑绅太傅四人王黼燕王俣越王偲郓王楷太保十一人蔡攸肃王枢至仪王防渡江后以用兵故绍兴十二年奉申王为太师张循王韩蕲王为太傅刘安城王为太保其后乾道初阳和王吴信王并为太傅绍兴初史会稽王为太师嗣秀王为太保盖自绍兴而后三公未尝备官焉朝野杂记】
