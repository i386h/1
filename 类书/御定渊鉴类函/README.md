　　钦定四库全书
　　御制渊鉴函序
　　朕几务余暇博渉艺林每揽一书必尽其全帙沉潜往复既得其始终条理精义之所存而文句英华亦常读之矣尝谓古人政事文章虽出于二然文章以言理政事则理之发迩而见远者也岂仅以其区区文句之间而可以自命为学术乎自六朝乃有书而尤盛于唐此岂非求之文句之间者哉虽然理之所寓于斯萃焉弗可废也昔者孔子之系易也曰方以聚又曰本乎天者亲上本乎地者亲下则各从其也于诸卦则曰其称名也小其取也大盖以天下古今事物之理毕具于易而易之为书因理象物因物徴辞以断天下之疑而成天下之务者各从其以明之然则书之作其亦不违于圣人立言之意欤书之最著者艺文聚北堂书钞初学记白帖杜氏通典宋明以来撰者寖广若博而不防简而能覈者抑亦鲜矣独俞安期唐函
　　颇称详括大抵祖述欧阳询之聚稍删存书钞初学记白帖通典而附益之安期明人也而曰唐函者以其皆唐辑也既缺宋以来书而唐以前亦有脱漏者爰命儒臣逖稽旁捜泝洄往籍网罗近代增其所无详其所略参伍错综以摛其异探赜索隐以约其同要之不离乎以相从而始备焉书成计四百五十卷夫自有书迄于今千有余年而集其大成可不谓斯文之少补乎学者或未能尽读天下之书观于此而得其大凡因以求尽其始终条理精义之所存其于格物致知之功修辞立诚之事为益匪浅尠矣

　　成命恭率翰詹诸臣商确凡例胪列区分益以他书句栉字比诸凡日星河岳部次贵于精详礼乐兵农制度求其明备以及禽鱼草木罔不捜罗道徳性情更加阐发踵孔门文学之科究歴代图书之府虽抱残守缺未能力致五车而落实取材窃拟目探二酉非徒撦割言工不数饾饤为富故首以音义明辨总载提纲而典故次之事对又次之单词只句有可采录另为一条不敢放失至于诗赋杂文则辨体标目删繁就简有节取之义焉譬夫翦裁在手集千腋而成裘组织任心防五丝以为采庶几方名象数幼学者展卷神开理干文条旷览者含毫色喜李商隐可无獭祭之劳虞秘书并省行厨之脚然诸【臣】既非成于一手【臣】等复荒谬而何知罣漏滋多鲁鱼不少置诸绨几之旁敢道笙簧六籍杂在芸编之末差言会萃百家恐犹学府之糠粃书仓之菅蒯伏冀
