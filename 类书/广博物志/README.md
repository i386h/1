　　钦定四库全书　　　　　子部十一
　　广博物志　　　　　　　类书类
　　提要
　　【臣】等谨案广博物志五十卷明董斯张撰斯张字遐周乌程人国子监生以洽闻周见有声于时所著述甚多此书名为广张华博物志而作其实分门事名目琐碎颇近后世类家与博物志原例不甚相合书中分大目二十有二子目一百六十有七所载始于三坟迄于隋代详畧互见未能首尾赅贯其徴引诸书皆标列原名缀于每条之末体例较善而中间亦有舛駮者如太平御览太平广记皆采摭古书原名具在乃斯张所引出自二书者徃徃但题御览广记之名而没所由来殊为不明根据又图经不言某州地志不言某代随意剽掇亦颇近于稗贩又汉杂事秘辛为杨慎伪作世所共知乃好异喜新杂然并载更不免疎于持择至若孔疏郑笺牵连满幅道经释典采録盈篇爱博贪多尤伤枝蔓然其搜罗既富唐以前之遗文坠简裒聚良多在明代类书中固犹为有资于淹雅者矣乾隆四十五年十二月恭校上

　　广博物志卷一
　　明　董斯张　撰
　　天道上【天　日　月】
　　天道尚右日月西移地道尚左水道东流人道尚中耳目役心心有四佐不和曰废地有五行不通曰恶天有四时不时曰凶天道曰祥地道曰义人道曰礼【周书】太初气之始也生于酉仲清浊未分也太始形之始也生于戌仲【八月酉仲为太初属雄九月戌仲为太始属雌】清者为精浊者为形也太素质之始也生于亥仲已有素朴而未散也三气相接至于子仲□判分离轻清者上为天重浊者下为地中和为万物【诗纬曰阳本为雄隂本为雌物本为防雄雌但行三节而雄合物防号曰太素也三未分别号曰浑沦○广雅】
　　天者何也天之为言镇也居髙理下为人镇也地者易也言飬万物怀任交易变化也【白虎通】
　　天以不见为地以不形为人以心腹为天奥西北郁化精也地奥黄泉隐魄荣也人奥思虑含至精也【太经】
　　天有七星地有七表天有四维地有四渎【河图括地象】天有精地有形天有八纪地有五岳【素问】
　　天以六六为节地以九九制防【上】
　　天道以九制地理以八制人道以六制【管子】
　　天左舒而起牵牛地右辟而起毕【尸子】
　　黄帝书曰天在地外水在天外水浮天而载地者也【隋书】
　　天象盖笠地法覆盘【周髀】
　　天青黑地黄赤天数之为笠也青黑为表丹黄为里以象天地之位【上】
　　春为苍天夏为昊天秋为旻天冬为上天【尔雅】
　　臣览太始天元册文丹天之气经于牛女戊分黅天之气经于心尾己分苍天之气经于危室栁鬼素天之气经于亢氐毕天之气经于张翼娄胃所谓戊己分者奎壁角轸则天地之门户也【素问】
　　九天一中天二羡天三从天四更天五晬天六廓天七咸天八沈天九成天【太元经】
　　中央曰钧天其星角亢氐东方曰苍天其星房心尾东北曰旻天其星箕斗牵牛北方曰天其星须女虚危营室西北方曰幽天其星东壁奎娄西方曰昊天其星胃毕西南方曰朱天其星觜嶲参东井南方曰炎天其星舆鬼栁七星东南方曰阳天其星张翼轸【淮南子】天有十端天地阴阳水土人金木火凡十端天亦有喜怒哀乐与人相副以类合之天人一也【春秋繁露】
　　四时者天之吏也日月者天之使也星辰者天之期也虹蜺彗星者天之忌也【淮南子】
　　天者诚日其德也日诚出诚入南北有极故莫弗以为法则天者信月其期也月信死信生终则有始故莫弗以为政天者明星其稽也列星不乱各以序行故大小莫弗以章天者因时其则也四时常相代而不干故莫弗以为必然天者一法其司也前后左右古今自如故莫弗以为常天诚信明因一不为众人易徳故莫能与争先【鹖冠子】
　　保章氏【官属如冯相氏○保官也世守天文之变】掌天文以志星辰日月之变动【天左转日月五星右行故变动而无常】以观天下之迁【天下灾祥祸福之迁遇者因可得而观焉】辨其吉凶以星土辨九州之地所封【星土星所主土也封犹界也】封域各有分星【角亢氐兖州房心豫州尾箕幽州斗牛女扬州虚危青州室壁并州奎娄胃徐州毕冀州觜参益州井鬼雍州栁星张三河翼轸荆州】以观妖祥【观彗孛客星所犯属何分野主何妖祥】以十有二嵗之相观天下之妖祥【嵗星在木则火为相在火则土为相在土则金为相在金则水为相在水则木为相五星顺度为祥流逆失度为妖】以五云之物辨吉凶水旱【云有五色以二分二至日观之青为防白为防赤为兵荒黒为水黄为丰】降丰荒之祲象【降其祲象于国使人预知而为备也】以十有二风察天地之和【十二风者风生于十二辰之位也天地之气和而成风和则无事】命乖别之妖祥【乖则异别则离此天地之不和而为妖祥也故命之以告于人焉】凡此五物者以诏救政访叙事【谋访天时所宜次序其事○周礼】
　　眡祲【官属如占夣】掌十煇之法【煇谓日之气煇光也】以观妖祥辨吉凶一曰祲【日旁有阴邪气相侵犯也】二曰象【阴气附日凝结而成象也】三曰镌【黒气如镌刺于日也】四曰监【抱日之气如冠珥也】五曰闇【方昼而暗也】六曰瞢【阴气蒙日光瞢然也】七曰弥【白虹弥天横气贯日也】八曰叙【片叚成列穿日有序也】九曰脐【防蝀升气于日旁也】十曰想【襍气以成形想也】○掌安宅叙降正嵗则行事嵗终则其事【上】
　　天地浑沌如鸡子盘古生其中万八千嵗天地开辟盘古在其中一日九变神于天圣于地天日髙一丈地日厚一丈盘古日长一丈如此万八千嵗天数极髙地数极深盘古极长后乃有三皇数起于一立于三成于五盛于七处于九故天去地九万里【三五厯记】
　　百世之后地髙天下不风不雨不寒不暑民复食土皆知其母不知其父如此千嵗之后而天可倚杵汹汹莫知始终【河图挺佐辅】
　　问云诸天衣服云何答曰如经说六欲界六天中皆服天衣飞行自在看之似衣光色具足不可以世间缯防比之色界诸天衣服虽号天衣衣如非衣其犹光明转胜转妙不可名也【法苑珠林】
　　忉利天衣重六铢炎摩天衣重三铢兠率陀天衣重一铢半化乐天衣重一铢他化自在天衣重半铢
　　问曰诸天住处其义云何答曰如婆沙论说天虽有三十二住处但有二十八重以彼四空絶离形报故无别处遍在欲色二界之中但随欲色二界众生成就四空无色业者随大乗说有色也其二十八重者谓湏弥山根从地上升去地四千由旬绕山纵广一万由旬是坚手天于中止住复上升一倍绕山八千由旬是彼持华鬘天于中止住复上一倍绕山四千由旬是彼常放逸天于中止住复上一倍绕山四千由旬是彼日月星宿天于中止住复上一倍绕山四千由旬是彼四天王天于中止住【其中由有七种金是四天王城聚落悉在其中】复上升四万由旬至须弥山顶纵广四万由旬其中有喜见城纵广一万由旬面别有其千门三十三天于中止住即从此山升虗空四万由旬有处如云七宝所成其犹大地是炎摩天于中止住复上一倍有地如云七宝所成是兠率陀天复上一倍有地如云七宝所成是化乐天复上一倍有地如云七宝所成是化自在天如是乃至色界究竟天皆悉有地如云七宝所成相去皆倍不烦具说依顺正理论云三十三天迷卢山顶其顶四面各二十千若据周回数成八万有余师说面各八十千与下际四边其量无别山顶四角各有一峰其髙广量各有五百有药乂神名金刚手于中止住守防诸天于山顶中有宫名善见面二千半周万逾缮那金城量髙一逾缮那半其地平坦亦真金所成俱用百一襍宝严餙地触柔輭如妬罗绵于践蹑时随足髙下是天帝释所都大城城有千门严饰壮丽门有五百青衣药乂勇健端严长一逾缮那量各严铠仗防守城门于其城中有殊胜殿种种妙宝具足庄严映蔽天宫故名殊胜面二百五十逾缮那周千逾缮那是谓城中诸可爱事城外四面四苑庄严是彼诸天其游戱处一众车苑谓此苑中随天福力种种车现二麤恶苑天欲战时随其所须甲仗等现三襍林苑诸天入中所玩皆同俱生胜喜四喜林院极妙欲尘襍类俱臻厯观无厌如是四苑形皆异方方周千逾缮那量居各有一如意池池面各五十逾缮那量各功德水弥满其中随欲四苑花鸟香林庄饰业果差别难可思议天福城外西南角有大善法堂三十三天时集辩论制伏阿素洛等如法不如法事起世经云佛告比丘以何因缘诸天防处名善法堂三十三天集防坐时于中唯论微细善语深义称量观察皆是世间诸胜要法真寔正理是以诸天称为善法堂又何因缘名波娄沙迦苑【隋言麤澁】三十三天王已入坐于贤及善贤二石之上唯论世间麤恶不善戱谑之语是故称波娄沙迦又何因缘名襍色车苑三十三天王入已坐于襍色善襍色二石之上唯论世间种种色相语言是故称为襍色车苑又何因缘名襍乱苑三十三天常以月八日十四日十五日放其宫内一切采女入此园中令与三十三天众合杂嬉戱不生障隔恣其欢娱爱天五欲具足功德游行爱乐是故诸天共称此园为襍乱苑又何因缘彼天有园名为欢喜三十三天王入其中已坐于欢喜善欢喜二石之上心爱观喜复爱极乐是故诸天共称彼园以为欢喜又何因缘名波利夜怛逻拘毗陀罗树下有天子住名曰未多日夜常以彼天种种五欲功德具足和合游戏爱乐是故诸天遂称彼树以为波利夜怛逻拘毗陀罗树
　　忉利天善见城周围四万十千由旬纯金为城城围绕髙十由旬城上埤堄髙半由旬门髙二由旬其外重门髙一由旬半十十由旬有一一门城之四面为千门楼是诸城门众宝所成种种摩尼之所严饰于大城四分之一中央金城帝释住处十二由旬有二门四面四百九十九门复有一小门凡五百门是城形相亦衞四兵栅堑榭池襍林宫殿作倡伎乐及诸外戏种种宝藏不可具述是城中央宝楼重阁名皮禅延多楼长五百由旬广二百五十由旬周围一千五百由旬其间四边却敌宝楼东边二十六所三面各二十五所凡一百一所一一却敌方二由旬周围八由旬其却敌上复有宝楼髙半由旬以为观望一一却敌有七女天一一女天有七采女楼阁之内有万七百房室一一房室有七天女一一天女采女亦七其天女者并是帝释正妃其外却敌及内诸房各四亿九万四千九百正妃三十四亿六万四千三百采女妃及采女合三十九万亿五万九千二百皮禅延多重阁最上当中央围室广三十由旬周围九十由旬髙四十五由旬是帝释所住之处并是琉璃所成众宝所填
