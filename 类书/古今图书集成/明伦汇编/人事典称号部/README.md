　称号部艺文
　　六一居士传　　　　　　　宋欧阳修六一居士初谪滁山自号醉翁既老而衰且病将退休于颍水之上则又更号六一居士客有问曰六一何谓也居士曰吾家藏书一万卷集录三代以来金石遗文一千卷有琴一张有棋一局而常置酒一壶客曰是谓五一尔奈何居士曰以我一翁老于此五物之间是岂不为六一乎客笑曰子欲逃名者乎而屡易其号此庄生所谓畏影而走乎日中者也余将见子疾走大喘渴死而名不得逃也居士曰吾固知名之不可逃然亦知夫不必逃也吾为此名聊以志吾之乐尔客曰其乐如何居士曰吾之乐可胜道哉方其得意于五物也太山在前而不见疾雷破柱而不惊虽响九奏于洞庭之野阅大战于涿鹿之原未足喻其乐且适也然常患不得极吾乐于其间者世事之为吾累者众也其大者有二焉轩裳珪组劳吾形于外忧患思虑劳吾心于内使吾形不病而已瘁心未老而先衰尚何暇于五物哉虽然吾自乞其身于朝者三年矣一日天子恻然哀之赐其骸骨使得与此五物偕返于田庐庶几偿其夙愿焉此吾之所以志也客复笑曰子知轩裳珪组之累其形而不知五物之累其心乎居士曰不然累于彼者已劳矣又多忧累于此者既佚矣幸无患吾其何择哉于是与客俱起握手大笑曰置之区区不足较也已而叹曰夫士少而仕老而休盖有不待七十者矣吾素慕之宜去一也吾尝用于时矣而讫无称焉宜去二也壮犹如此今既老且病矣乃以难强之筋骸贪过分之荣禄是将违其素心而自食其言宜去三也吾负三宜去虽无五物其去宜矣复何道哉熙宁三年九月七日六一居士自传　　一日子解　　　　　　　　　　赵儒　予感岁暮乃为此解冀良久相与砥砺不我弃也客有问于一日子曰子欲志道胡谓一日也岂不闻道体无外人心易私非道远人人自远而虽终身之未逮岂一日之可期一日子曰千里匪远跬步匪近千钧匪重锱铢匪轻视之如一乃得其情岂不闻志高者易荒名盛者易倾与其悠悠以卒岁孰若竟日而冰兢客曰噫噫为山九仞功亏一篑凿井未泉犹以为弃日月逝焉圣门是愧杯水虽勤车薪犹炽如子所云不几于废道乎哉一日子曰以昔视今犹今视昔亿万斯年犹之一日苟今日之未力又来日之敢必客曰子之言吾知之矣抔土不让细流不择积小高大而无间隙日复一日持久乃益一日子曰是道也先意而为之吾弗敢也不耕而获不灾而畬有望斯妄无为斯为厥心憧憧是为自欺客曰子之言得矣盍尽言之一日子曰客不闻乎孔子有言一日用力力无不足朝闻乎道夕死可矣一日克复天下归之吾知有今日不知有来日至于来日视为今日泰山在前而不见其形震惊在侧而不闻其威吾谓一日如是而已客曰闻命矣遂谓一日子云

宋书王敬弘传敬弘形状短小而坐起端方桓元谓之弹□八势所居舍亭山林涧环周备登临之美时人谓之王东山王景文传景文伯父智少简贵有高名高祖甚重之与刘穆之谋讨刘毅而智在焉他日穆之白高祖曰伐国重事也公云何乃使王智知高祖笑曰此人高简岂闻此辈论议其见知如此郑鲜之传鲜之历御史中丞尽心高祖高祖少事戎旅不经涉学及为宰相颇慕风流时或言论人皆依违之不敢难也鲜之难必切至未尝宽假须要高祖辞穷理屈然后置之高祖或有时惭恧变色动容既而谓人曰我本无术学言义尤浅比时言论诸贤多见宽容惟郑不尔独能尽人之意甚以此感之时人谓为格沈庆之传庆之患头风好着狐皮帽群蛮恶之号曰苍头公谢弘微传弘微叔父混名知人少所结纳惟与族子灵运瞻曜弘微并以文义赏会瞻等才辞辩富弘微每以约言服之混特所敬贵号曰微子弘微除镇西谘议参军太祖即位为黄门侍郎与王华王昙首殷景仁刘湛等号曰五臣何承天传承天除著作佐郎撰国史承天年已老而诸佐并名家年少颍川荀伯子□之常呼为你母承天曰卿当云凤凰将九子你母何言邪周续之传续之年八岁丧母哀戚过于成人奉兄如事父豫章太守范□于郡立学招集生徒远方来者甚众续之年十二诣□受业居学数年通五经并纬候名冠同门号曰颜子谢方明传刘穆之权重当时朝野辐辏不与穆之相识者惟有混方明郗僧施蔡廓四人而已穆之甚以为恨方明廓后往造之大说白高祖曰谢方明可谓名家驹直置便自是台鼎人无论复有才用张卲传卲子敷字景引生而母亡年数岁问知之虽童蒙便有感慕之色至十岁许求母遗物而散施已尽唯得一扇乃缄录之每至感思辄开笥流涕见从母悲感呜咽性整贵风韵端雅好元言善属文初父卲使与南阳宗少文谈系象往复数番少文每欲屈握麈尾叹曰吾道东矣于是名价日重武帝闻其美召见奇之曰真千里驹也王元谟传元谟为金紫光禄大夫领太常及建明堂以本官领起部尚书又领北选孝武狎侮群臣随其状貌各有比类多须者谓之羊颜师伯缺齿号之曰齴刘秀之俭吝呼为老悭黄门侍郎宗灵秀体肥拜起不便每至集会多所赐与欲其瞻谢倾踣以为欢笑又刻木作灵秀父光禄勋叔献像送其家厅事柳元景垣护之并北人而元谟独受老伧之目凡所称谓四方书疏亦如之尝为元谟作四时诗曰菫荼供春膳粟浆充夏瓟酱调秋菜白醝解冬寒又宠一昆仑奴子名曰主常在左右令以杖击群臣自柳元景以下皆罹其毒始安王休仁传废帝狂悖无道诛害群公忌惮诸父并囚之殿内殴捶凌曳无复人理休仁及太宗山阳王休佑形体并肥壮帝乃以竹笼盛而称之以太宗尤肥号为猪王号休仁为杀王休佑为贼王以三王年长尤所畏惮故常录以自近不离左右东海王袆凡劣号为驴王沈约自序沈正字元直淹详有器度美风姿善容止好老庄之学弱冠辟从事宗人光禄大夫演之称之曰此宗中千里驹也袁粲传粲徙尚书令与齐王褚渊刘秉入直平决万机时谓之四贵谢灵运传灵运免官东还与族弟惠连东海何长瑜颍川荀雍太山羊璇之以文章赏会共为山泽之游时人谓之四友莲社高贤传周续之雁门人十二诣范□受业通五经五纬时号十经童子异苑有人姓刘在朱方人不得共语若与之言必遭祸难或本身死疾惟一士谓无此理偶值人有屯塞耳刘闻之忻然而往自说被谤君能见明答云世人雷同亦何足恤须臾火燎资蓄服玩荡尽于是举世号为刘鸺鹠脱遇诸涂皆闭车走马掩目奔避刘亦杜门自守岁时一出则人惊散过于见鬼南齐书高祖本纪休范破平太祖振旅凯入百姓缘道聚观曰全国家者此公也太祖与袁粲褚渊刘秉引咎解职不许迁散骑常侍中领军都督南兖徐兖青冀五州军事镇军将军南兖州刺史持节如故进爵为公增邑二千户太祖欲分其功请益粲等户更日入直决事号为四贵秦时有太后穰侯泾阳高陵君称为四贵至是乃复有焉虞玩之传孔□字世远与王俭至交升明中为齐台尚书仪曹郎太祖谓之曰卿仪曹才也俭为宰相□尝谋议帷幕每及选用颇失乡曲情俭从容启上曰臣有孔□犹陛下之有臣也永明中为太子家令卒时呼孔□何宪为王俭三公长沙威王晃传晃便弓马多从武容熏赫都街时人为之语曰焕焕萧四伞桂阳王铄传鄱阳王锵好文章铄好名理时人称为鄱桂萧坦之传坦之肥黑而无须语声嘶时人号为萧□檀超传超入为殿中郎兼中书郎零陵内史征北骠骑记室国子博士兼左丞超嗜酒好言咏举止和靡自比晋□超为高平二超谓人曰犹觉我为优也张岱传岱少与兄太子中舍人寅新安太守镜征北将军永弟广州刺史辨俱知名谓之张氏五龙褚炫传炫迁中书侍郎司徒右长史升明初炫以清尚与刘俣谢朏江入殿侍文义号为四友何戢传戢美容仪动止与褚渊相慕时人呼为小褚公王秀之传秀之祖裕性贞正徐羡之傅亮当朝裕不与来往及致仕隐吴兴与子瓒之书曰吾欲使汝处不竞之地瓒之历官至五兵尚书未尝诣一朝贵江湛谓何偃曰王瓒之今便是朝隐及柳元景颜师伯令仆贵要瓒之竟不候之至秀之为尚书又不与王俭款接三世不事权贵时人称之陆澄传澄当世称为硕学读易三年不解文义欲撰宋书竟不成王俭戏之曰陆公书厨也臧荣绪传荣绪常以宣尼生庚子日陈五经拜之自号被褐先生初荣绪与关康之俱隐在京口世号为二隐杜京产传永明中会稽钟山有人姓蔡不知名山中养鼠数十头呼来即来遣去便去言语狂易时谓之谪仙徐伯珍传伯珍隐处豫章王辟议曹从事不就家甚贫窭兄弟四人皆白首相对时人呼为四皓江祏传东昏即位祏兄弟更直殿内永元元年领太子詹事刘暄迁散骑常侍右卫将军祏兄弟与暄及始安王遥光尚书令徐孝嗣领军萧坦之六人更日帖敕时呼为六贵谢超宗传司徒褚渊送湘州刺史王僧虔阁道坏坠水仆射王俭尝牛惊跣下车超宗抚掌笑戏曰落水三公堕车仆射前后言诮稍布朝野梁书武帝本纪竟陵王子良开西邸招文学高祖与沈约谢朓王融萧琛范云任昉陆倕等并游焉号曰八友贺琛传琛为通直散骑常侍领尚书左丞并参礼仪事琛前后居职凡郊庙诸仪多所创定每见高祖与语常移晷刻故省中为之语曰上殿不下有贺雅琛容止都雅故时人呼之范岫传岫迁太子家令文惠太子之在东宫沈约之徒以文才见引岫亦预焉岫文虽不逮约而名行为时辈所与博涉多通尤悉魏晋以来吉凶故事约常称曰范公好事该博胡广无以加南乡范云谓人曰诸君进止威仪当问范长头以岫多识前代旧事也王茂传茂字休远太原祁人也祖深北中郎司马父天生宋末为列将于石头克司徒袁粲以勋至巴西梓潼二郡太守上黄县男茂年数岁为大父深所异常谓亲识曰此吾家之千里驹成门户者必此儿也张率传率建武三年举秀才除太子舍人与同郡陆倕幼相友狎常同载诣左卫将军沈约适值任昉在焉约乃谓昉曰此二子后进才秀皆南金也卿可与定交由此与昉友善谢几卿传几卿幼清辨当世号曰神童
