　闺节部列传一
　　周
　　卫寡夫人
按刘向列女传卫寡夫人者齐侯之女也嫁于卫至城门而卫君死保母曰可以还矣女不听遂入行三年之丧毕弟立请曰卫小国也不容二庖请愿同庖终不听卫君使人訴于齐兄弟齐兄弟皆欲与君使人告女女终不听乃作诗曰我心匪石不可转也我心匪席不可卷也厄穷而不悯劳辱而不苟然后能自致也言不失也然后可以济难矣诗曰威仪棣棣不可选也言其左右无贤臣皆顺其君之意也君子美其贞一故举而列之于诗也　　宋恭伯姬
按刘向列女传伯姬者鲁宣公之女成公之妹也其母曰缪姜嫁伯姬于宋恭公恭公不亲迎伯姬迫于父母之命而行既入宋三月庙见当行夫妇之道伯姬以恭公不亲迎故不肯听命宋人告鲁鲁使大夫季文子如宋致命于伯姬还复命公享之缪姜出于房再拜曰大夫勤劳于远道辱送小子不忘先君以及后嗣使下而有知先君犹有望也敢再拜大夫之辱伯姬既嫁于恭公十年恭公卒伯姬寡至景公时伯姬尝遇夜失火左右曰夫人少避火伯姬曰妇人之义保傅不俱夜不下堂待保傅来也保母至矣傅母未至也左右又曰夫人少避火伯姬曰妇人之义傅母不至夜不可下堂越义而生不如守义而死遂逮于火而死春秋详录其事为贤伯姬以为妇人以贞为行者也伯姬之妇道尽矣当此之时诸侯闻之莫不悼痛以为死者不可以生财物犹可复故相与聚会于澶渊偿宋之所丧春秋善之君子曰礼妇人不得傅母夜不下堂行必以烛伯姬之谓也诗云淑慎尔止不愆于仪伯姬可谓不失仪矣　　鲁寡陶婴
按刘向列女传鲁寡陶婴者鲁陶门之女少寡养幼孤无强昆弟纺绩为产鲁人或闻其义将求焉婴闻之恐不得免作歌明己之不二也其诗曰悲黄鹄之早寡兮七年不双鸳颈独宿兮不与众同夜半悲鸣兮想其故雄天命早寡兮独宿何伤寡妇念此兮泣下数行呜呼哀哉兮死者不可忘飞鸟尚然兮况于贞良虽有贤雄兮终不重行鲁人闻之曰斯女不可得矣遂不敢复求婴寡终身不改君子谓陶婴贞一而思诗云心之忧矣我歌且谣此之谓也　　楚昭贞姜
按刘向列女传贞姜者齐侯之女楚昭王之夫人也王出游留夫人渐台之上而去王闻江水大至使使者迎夫人忘持其符使者至请夫人出夫人曰王与宫人约令召宫人必以符今使者不持符妾不敢从使者行使者曰今水方大至还而取符则恐后矣夫人曰妾闻之贞女之义不犯约勇者不畏死守一节而巳妾知从使者必生留必死然弃约越义而求生不若留而死耳于是使者取符则水大至台崩夫人流而死王曰嗟夫守义死节不为苟生处约持信以成其贞乃号之曰贞姜君子谓贞姜有妇节诗云淑人君子其仪不忒此之谓也　　楚白贞姬
按刘向列女传贞姬者楚白公胜之妻也白公死其妻纺绩不嫁吴王闻其美且有行使大夫持金百镒白璧一双以聘焉因以辎軿三十乘迎之将以为夫人白妻辞之曰白公生时妾幸得充后宫执箕掌衣履拂枕席托为妃匹白公不幸而死妾愿守其坟墓以终天年今王赐金璧之聘夫人之位非愚妾之所愿闻也且夫弃义从欲者污也见利忘死者贪也夫贪污之人王何以为哉妾闻之忠臣不借人以力贞女不假人以色岂独事生若此哉于死者亦然妾既不仁不能从死今又去而嫁不亦太甚乎遂辞聘而不行吴王贤其守节而有义号曰贞姬君子谓贞姬廉洁而诚信夫任重而道远仁以为己任不亦重乎死而后已不亦远乎诗云彼美孟姜德音不忘此之谓也　　梁寡高行
按刘向列女传高行者梁之寡妇也其为人荣于色而美于行夫死早寡不嫁梁贵人争欲取之不能得梁王闻之使相聘焉高行曰妾夫不幸早死先狗马填沟壑妾守养其孤幼曾不得专意贵人多求妾者幸而得免今王又重之妾闻妇人之义一往而不改以全贞信之节忘死而趋生是不信也见贵而忘贱是不贞也弃义而从利无以为人乃援镜持刀以割其鼻曰妾已刑矣所以不死者不忍幼弱之重孤也王之求妾者以其色也今刑余之人殆可释矣于是相以报王大其义高其行乃复其身尊其号曰高行君子谓高行节礼专精诗云谓予不信有如皎日此之谓也　　秦
　　巴蜀寡妇
按史记货殖列传巴蜀寡妇清其先得丹穴而擅其利数世家亦不訾清寡也能守其业用财自卫不见侵犯秦皇帝以为贞妇而客之为筑女怀清台　　后汉　　陈仲妻张氏
按皇甫谧列女传安定陈仲妻者同郡张叔明之妹名芝字李张年十四适仲&#年而寡执节不嫁叔明从军芝与二嫂没贼恐见侵掠而相谓曰妇人以不污身为高不亏节为美岂可委身待辱哉于是自刺二嫂既死芝独不死叔明言于将军耿弇耿弇以附马负芝芝曰女子未亡之余污将军服乘不可也弇奇其言更以他马负芝至营为致医药因乃得全郡表其闾九十寿终　　刘长卿妻
按后汉书列女传沛刘长卿妻者同郡桓鸾之女也生一男五岁而长卿卒妻防远嫌疑不肯归宁儿年十五又夭殁妻虑不免乃豫刑其耳以自誓宗妇相与愍之共谓曰若家殊无它意假令有之犹可因姑娣妹以表其诚何贵义轻身之甚哉对曰昔我先君五更学为儒宗尊为帝师五更以来历代不替男以忠孝显女以贞顺称诗云无忝尔祖聿修厥德是以豫自刑剪以明我情沛相王吉上奏高行显其门闾号曰行义桓县邑有祀必膰焉　　唐姬
按袁宏后汉记弘农王唐姬故太守唐瑁女也王薨父欲嫁之不从及关中破为李傕所掠不敢自说也傕欲妻之姬弗听尚书贾诩闻之以为宜加爵号于是近置于园拜为弘农王妃　　留子直妻
按皇甫谧列女传留子直妻者历阳人汉末扰攘随夫之从父客居豫章从父通郡牧族之妻年少有色太守客请以为妻守死不从十余日客以还太守将杀之临死不变口无言郡吏及客怜之更还救既得活乃自割耳久之太守闻其夫在遂还其妻　　夏文生妻刘氏
按皇甫谧列女传梁夏文生妻者沛国刘景宾之女名娥生一女而寡娥誓不再嫁父以配同郡衡氏逼迫入门娥谓衡氏曰妾闻妇人不改嫁越义失节妾所不为君可见遣衡氏曰相取有媒礼何遣之有衡氏妻服未阕娥因数之曰君衰麻在身犯礼纳室虽颜之厚奈相鼠何妾必死不为君妻相留不知辱乎奋衣而出衡氏不敢强留父复以许临淮倪氏强扶上船娥以刀割耳鼻曰所以不死者老姑在堂孤女尚幼故耳执义终身　　公孙去病妻戴氏
按皇甫谧列女传沛公孙女病妻者同郡戴元世之女既嫁久而无子谓其夫曰妾不才得奉巾栉历年无嗣礼有七出请愿受诀矣其夫不许复进曰福莫大于昌炽祸莫大于绝嗣君不忍见遣当更广室夫复不肯夫死服除父母欲嫁之女遂操刀割鼻郡表其闾　　赵嵩妻张氏
按皇甫谧列女传汉中赵嵩妻者同郡张氏之女也字礼修遭贼嵩死君难礼修以碧涂面乱发称病怀刀在身意气烈决贼不迫也叔父矜其年少又世方丧乱欲更嫁礼修慷慨以死为誓　　孙奇妻范氏
按皇甫谧列女传吴孙奇妻者广陵范慎女名姬十八配奇一年而奇亡慎以姬少寡无子趣还其家姬不肯归迎以父命迫之姬遂操刀割耳及鼻曰父迎我者不过以我年少色美今已残矣行将焉之于是迎者空反　　华穆妻刘氏
按皇甫谧列女传丹阳华穆妻者下邳刘芳之女字桃树生一男而夫早亡吴知名之士家将娶许焉桃树闻之乃操刀割耳其子又亡桃树安身守正动不失礼　　沈伯阳妻顾氏
按皇甫谧列女传吴沈伯阳妻者顾文选之女字昭君早寡无嗣尽礼供养其父阴许人姑闻而哭昭君乃引刀剪发割两耳以明其志　　朱旷妻罗静
按皇甫谧列女传丹阳罗静者广德罗勤之女为同县朱旷所聘婚未成勤遇疫疾丧没邻比断绝旷触冒经营寻复病亡静感其义遂誓不嫁与弟妹共居求者过十余志无倾移有杨祚者多将人众自往纳币静乃逃窜祚劫其弟妹静惧为所害乃出见之曰實感朱旷为妾父而死是以托身亡者自誓不二辛苦之人愿君哀而舍之如其不然请守之以死祚乃舍之静守纯固年六十余卒　　景奇妻罗氏
按皇甫谧列女传蜀景奇妻者罗氏之女字贡罗奇亡无嗣贡罗专心供养父清以许同郡宰诗贡罗与父母书陈其情志历年不归后清使诗白州告县发遣贡罗乃由径道诣州自诉言意慷慨请死不从州嘉而许焉贡罗恐诗于道路迫&#乃请吏兵自卫还家执义终身　　相登妻周氏
按皇甫谧列女传犍为相登妻者周氏之女名度适登一年而寡牢令吴厚因人问度心执匪石引刀截发县长史复遣媒欲聘度曰前已断发谓足表心何&#后有斯言哉取刀欲割鼻左右救止表其闾　　冯季宰妻李氏按皇甫谧列女传广汉冯季宰妻者李氏之女名珥字进娥早寡无嗣奉养继姑及宰兄显守心纯固以义自防珥母愍之其姑阴有所许珥断发自誓遂乞养男女各一乡人称之　　王辅妻彭氏
按皇甫谧列女传广汉王辅妻彭氏之女也名非辅游学数年遂卒京师迎丧葬讫事姑孝敬弥笃非叔公以许苏孟非叩心泣血诉情九族犹不见听乃剪发诣府乞终供养遂乞养子静居年逾七十而卒　　公乘士会妻张氏按益都耆旧传蜀郡广都公乘士会妻者同县张氏女也会早卒年壮无嗣欲有问者亲戚将以许之发愤慷慨断发割耳事姑尽礼肃恭供养养族子以承宗祀　　廖伯妻因氏
按益都耆旧传广汉廖伯妻者同县因氏女也名纪年十六适伯伯早卒既性聪敏达于诗书女传进退闲暇又有美色见者贪之割面告诚以全其节曰求生害仁仁者不为纪生见礼义岂独使古人擅名者哉因作诗三章以风父母而举县嘉其才丽媒介滋繁遂援刀断指以明其志　　虞某妻邵氏
按邵氏家传虞建武都尉邵夫人字义姬鸿胪之第二女夫人少而寡虞氏及夫人之宗哀夫人辛苦欲更为图婚然重夫人宿操虑不可以非礼逼亦知夫人先佩刀誓以必死故不敢生意夫人自以虞氏凶短继世无子常独处一室绝书学非祭祀坟墓不出纺绩辄货以供祭称其多少不求丰厚　　杨凤珪妻陈氏
按益都耆旧传犍为杨凤珪妻者蜀郡临邛陈氏女也名姬珪早亡时姬产子适生六月躬丧事育幼孤三年丧讫兄弟宗亲哀其子少年壮谋议更配以许蜀中豪姓姬闻仰天叹息引刀割咽几死于是九族惊愕遂敬从其节　　便敬妻王氏
按益都耆旧传广汉新都便敬妻者同县王氏女也名和年十七适敬敬亡和育养遗孤阖门守节不随宗家宴乐嘉会居理甚修蜀郡何玉因谋问和兄着取和遂相听许着深晓曰夫死子小经有改图加贫衰无以自立何氏公族必据福祚和自陈说计断决分守全孤弱辞言未讫慷慨涕泪哀动左右然着终受玉币因欲迫胁和乃断耳示着以信着不见听请以死谢举宗敬重哀其大义　　周缮纪妻曹氏
按益都耆旧传犍为南安周缮纪妻者同县曹氏女也名禁字敬姬年十七适周氏二年而夫亡时禁怀妊数月后产子元余丧事阕遂移居依父母必欲守义育养孤弱父愍其年少子稚默以许同县狐宾遣车马衣服来欲迎禁父乃告禁勃然作色凄怆言曰依近父母本不图此因流涕慷慨乃自投舍后流水于是举家竞赴救出而气息已绝积二日一夜乃复苏息二亲由是知其至诚谢宾解婚禁歔欷长叹乃更将子还依夫弟居止洁身执操非礼不动　　陈顺谦
按汉中士女志陈顺谦成固人也适邓令曹宁十九寡居长育遗孤八十余卒兄弟陈规著书述之　　魏　　庞林妻习氏
