　　名　宋书明月珠王者不尽鳞介之物则出　淮南子明月之珠生于蠬蜄　汉书校猎赋剖眀月之珠胎师古注珠在蛤中若怀姙然故谓之胎也　裴氏广州记鲸鲵目即明月珠故死不见有目精　汉书武帝时使人入海市明月大珠至围二寸已下　崔豹古今注章帝元和元年明珠出馆陶大如李有明耀三年明月珠出豫章海昏大如鸡子围四寸八分　神异经西北荒中二金阙高百丈有明月珠径三寸光照千里　善信经明月珠摩尼珠多在龙脑中　北史南天竺出摩尼珠　法珠林长阿含经云郁单国有自然釡鍑有摩尼珠名曰熖光置于鍑下饭熟光灭不假樵火不劳人功　事物绀珠摩尼珠又名如意珠状如芥粟能变水色随人以何色物裹之水即如所裹物色　山堂肆考虚空藏菩萨身二十由旬顶上如意珠作紫金色无涯际　杂宝藏经佛言如意珠是摩竭大鱼脑中出鱼身长二十八万里此珠名曰金刚坚也常出一切宝物衣服饮食随意皆得得此珠者毒不能害火不能烧北史倭有如意宝珠其色青大如鸡卵夜则有光云鱼眼睛也　杜阳编唐肃宗时李氏尼真如得天帝所授八宝献于天子一曰如意宝珠其形正圆大如鸡卵宝光莹澈置之堂中明如满月　黄帝内传女授帝九明之珠　小説孔子得九曲珠欲穿不得遇二女教以涂脂于线使蚁通焉　葛洪西京杂记高祖斩白蛇剑上有七彩珠为饰　捃拾菁华王武子以九芒珠穿为纫索编华架用之每月洗以鲤鱼涎　五侯鲭七曜珠漳州龙溪所贡珠围三寸七分中有小珠七颗如七曜次如七曜者甚多　王子年拾遗记舜葬苍梧之野有鸟名凭霄雀衔青砂珠积成垄阜名曰珠丘其珠轻细风吹如尘起名曰珠尘　道籥西王母献穆王以云泽之珠又名却病珠设有人病在心则东向含之肝北肾西肺则俯身向地脾则南也无不应时愈　拾遗记燕昭王坐握日之台时有黑鸟白头集王之所衔洞光之珠圆径一尺色黑如漆悬照于室内百神不能隐其精灵此珠出阴泉之底有黑蚌千嵗一生珠珠渐轻细昭王常懐此珠当隆暑之月体自轻凉号曰销暑招凉之珠也　述异记汉武时水木之精献洞穴珠生于赤蚌其珠径寸　郭宪洞防记太初三年起甘泉望风台台上得白珠如花一枝帝以锦盖覆之如照月矣因名照月珠以赐董偃盛以琉璃之筐　洞防记东方朔西过洞壑得静海逰珠　飞燕外传真腊夷献不夜珠光彩若月照人无妍丑皆美艳帝以珠赐婕妤婕妤号为枕前不夜珠为后夀　集微五鹿充宗赠成子衍济之珠一名记时珠投之盘中滚滚不定至一时尽略定少选成子读书每以记时不差一遍　广异记则天时西国献青泥珠一枚珠拇指微青后不知贵以施西明寺僧布金刚额中后有胡人云西国有青泥泊多珠珍宝但苦泥深不可得若以此珠投泊中泥悉成水其宝可得则天因宝持之至宗时犹在　记闻唐大安国寺睿宗尝施一宝珠状如片石赤色夜则微光光高数寸有西域胡人见之曰此水珠也每军行休时掘地二尺埋珠其中水泉立出可给数千人以钱四千万贯买之去　杜阳杂编开元初罽宾国贡上清珠光明洁白可照一室视之则有仙人玉女云鹤绛节之象动摇于其中及代宗即位宝库中徃徃有神光异气　开元遗事开元中张説为相有人惠一珠绀色有光名记事珠或有遗忘即玩此珠心神顿悟　孔六帖大厯中林邑国献灵珠五色红而光又名诘多珠同菖蒲煮食即大如鸡卵可数日不饥　杜阳杂编顺宗即位嵗拘弭国贡履水珠色黑类铁大如鸡卵其上鳞皱其中有窍云持入江海中可行于洪波之上遂命善浮者以五色丝贯之系于左臂遣入龙池其人则步骤于波上若在平地亦潜于水中良乆复出而徧身略无沾湿上竒之至长庆中嫔御试弄于海池遂化为黑龙入于池内俄而云烟暴起不复追讨　异闻志成武丁进上以沙摩掩日之珠帝每集宫妃宴防则悬此珠于中庭一庭日色寒于夜月欢饮乆之恒忘白昼一小宫人爱眠当昼起拭目曰如何夜半鸟啼如此唤人醒也帝闻之大笑南越志士人宋述家有一珠大如鸡卵微绀色莹彻如水手持之映空而观则末底一防凝翠其上色渐浅若回转则翠处常在下不知何物或谓之滴翠珠　致虚杂俎郎玉嗜酒而家赤贫遇仙女于嵩山之中投以一珠曰此醉龙珠也诸龙含之以代酒味逾若下　宣室志冯翊严生家于汉南逰岘山得一珠如弹丸黑色而有光视之莹彻如冰焉以示西国胡人曰此是清水珠也即命注浊水于缶以珠投之俄而淡然清彻矣胡人以三十万货之　南墅闲居録苏子瞻尝言其先祖光禄云有一书生昼坐檐下见大蜂触蛛网相螫乆之蛛堕地起视之已化为小石矣书生异而之因置衣带中一日过市遇蛮贾数辈视书生愕眙揖曰愿见神珠书生戏以带中石示之羣贾相顾喜曰此破雾珠蛮人至海上采珠宝常以雾暗为苦有此珠雾即自开因以宝货易之值数千缗　元史成宗时西域贾人有以押忽大珠求售者曰含之可不渴熨面可使目有光　戒庵漫笔治年间欲分封诸王取珠于广得一珠甚大半黑如墨截然平分希世之宝也名天地分　华严经海底有四大宝珠性极猛能缩百川之水一曰日藏二曰离润三曰火光四曰尽无余　词林海错江歴珠名即江骊也汲冡周书正西昆仑请以江歴龙角为献　刘郁西使记西域报达国所产大珠曰太嵗弹郭义恭广志有珠称夜光大径寸有至圆珠置于地终日不停有石珠铸石为之一名朝珠今尚方名以甲乙为次第　通雅古有辟寒珠辟尘珠夜光照乗大者径寸或出于龙鱼异物腹中非独出于蚌也　陆佃埤雅龙珠在颔蛟珠在皮蛇珠在口鼈珠在足鱼珠在眼蚌珠在腹　梁四公记三珠其一是天帝如意珠之下者其二是骊龙珠之中者七珠二是虫珠五是海蚌珠人间之上者杂珠是蚌蛤等珠不如火珠之贵如意珠上者夜光照四十余里中者十里下者一里光之所及无风雨雷电水火刀兵诸毒厉骊珠九色上者夜光百步中者十步下者一室光之所及无蛇虺虫豸之毒虫珠七色而多赤其虫六足二目目当其陷处有凹如銕鼻蚌珠五色皆有夜光及数尺无瑕者为上其有瑕者为下蚌珠生于蚌腹与月盈亏蛇珠所致隋侯哙参即其事也　述异记南海有珠即鲸目瞳夜可以鉴谓之夜光凡珠有龙珠龙所吐也蛇珠所吐也南海俗云珠千枚不及一玫瑰言蛇珠贱也越人俗云种千亩木奴不如一龙珠吴越间俗説明珠一斛贵如王者合浦有珠市　庄子河上翁子得千金之珠其父曰此珠在骊龙颔下汝得之必遭睡也　幽明録洛下有田穴妇有杀夫推而下之得一穴所至告饥有长人指中庭栢树下有一羊令跪捋羊须三捋得三珠食焉复寻穴出还洛问张华云羊为痴龙初一珠食之天地等夀次者延年后者疗饥而已　博物志鲛人水底居出向人间住积日卖绡临去后主人索器泣而成珠满盘以与主人　拾遗记瀛州有鸟如凤身绀翼丹名藏珠每鸣翔则吐新珠累斛仙人以饰仙裳轻而耀于日月也搜神记有鹤为弋人所射穷而归哙参参养疗治疮愈而放之后鹤夜到门外雌雄各衔明珠以报参焉洞防记赤松子以凤葵草饵之三嵗乗黄蛇入水得
　　黄珠一枚色如真金或言是黄之卵故名珠亦曰销疾珠语曰宁失千里驹不失黄蛇珠　搜神记隋侯行见大蛇被伤而治之后衔珠以报其珠径寸纯白夜有光明如月之照一名隋侯珠一名明月珠一名灵蛇珠　洞防记蛇珠出涂云国有青灵蛇产珠色光白如琼琰之　稗史编栁州种甘堂顷年夜有光出柱上蠧穴中满堂如月太守知异物集吏卒持斧铤刺之有声砉然破柱乃大蜈蚣长竟柱破脑得珠如鵞卵圆莹光彩无价宝也　广异记狐口中媚珠若能得之当为天下所爱以绳缚狐四足以大笼罩其上用小缾口窄者埋地中令口与地齐以两胾猪肉炙于缾中狐爱炙而不能得但以口属缾候炙冷复下两胾狐涎沫乆之炙与缾满狐乃吐珠而死珠状如棊子通圆而洁北户録鲎珠南人或带或磨饮之云利市　李材解酲语至元间马八儿国入贡虾啰百颗形如珠而成龙纹大者过于弹丸国有虾濑隐沙中常抱珠戏于濑上土人俟其去取之　博物志五月五日取青蛉头正中门埋皆成青珠　拾遗记员丘穴洞中有细珠如流沙可穿而结因用为佩　西京杂记霍光妻遗淳于衍走珠一琲　珍玩考走珠置之于地能自走世谓之走珠下帷短牒力珠如龙眼大含之多力可以挽象尾使之倒行　唐书婆利国出火珠大者如鸡卵圜白照数尺日中以艾借珠辄火出　珍玩续考辟珠大者如指顶次如菩提子次如黍粟质理坚重如贝辟铜铁者铜铁不能损辟竹木者竹木不能损犯以他物即毁矣常附胎于椰子槟榔果谷之实之内通谓之圣銕岛夷能辨之故以为竒宝也　五侯鲭江珠即琥珀千年茯苓所化　徐衷南方草物状白蚌珠长三寸半在涨海中其一寸五分有光色一旁小形似覆釜为第一其一寸三分虽有光色形不圆正为第二　列仙传朱仲者防稽贩珠人也汉高后时募三寸珠仲诣阙上之珠好过度赐五百金鲁元公主私以七百金从仲市得四寸珠景帝时复献三寸珠数十枚　漂粟手牍珠三寸者谛视之有花影层层在内一里之内所种花木必现于中顔色宛然变幻万端第非时之木不见耳　列仙传郑交甫至汉臯台下见二女佩两珠大如荆鸡卵二女解与之既行反顾二女不见佩珠亦失　魏畧夫余出珠大如酸　沈存中笔谈嘉祐中州有一珠甚大初出于天长县陂泽中后转入甓社湖后又入新开湖中九十余年常常见之珠大如拳烂然不可视十余里间林木皆有影如日初照　沈怀远南越志珠有九品大五分以上至一寸八九分尢为入品有光彩一边小平似覆釜者名珰珠珰珠之次为走珠走珠之次为滑珠滑珠之次为磥砢珠磥砢珠之次为官雨珠官雨珠之次为税珠税珠之次为防苻珠　事物绀珠白影珠出侯道昌头砚中患目煮珠水洗之即愈　鱼豢魏畧大秦国出明珠夜光珠真白珠　南越志海中有大珠明月珠水晶珠　常璩华阳国志广阳县山出青珠永昌郡博南县有光珠穴出光珠有黄珠白珠青珠碧珠庄子黄帝逰赤水之北登昆仑之丘遗其珠　广博物志汉昭灵后名含始逰浴池有玉鸡衔赤珠出刻曰玉英吞此者王含始吞之生高祖　北史室韦俗爱赤珠为妇人饰穿挂于颈以多为贵女不得此乃至不嫁物相感志帝青珠大秦以来为絶价至珍其光华
