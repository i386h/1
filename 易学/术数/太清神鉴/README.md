　　提要
　　太清神鉴六卷　　　　术数类五【命书相书之属臣】等谨案太清神鉴六卷旧本题后周王朴撰乃専论相法之书也考朴事周世宗为枢密使世宗用兵所向克捷朴之筹画为多欧阳修新五代史称朴为人明白多材智非独当世之务至于隂阳律法莫不通焉薛居正旧五代史亦谓朴多所该综星纬声律莫不毕殚然皆不言其善于相法且此书前有自序称离林屋洞下山三载徧搜古今集成此书考朴家世东平入仕中朝逰迹未尝一至江左安得有隠居林屋山事其为依托无疑盖朴以精通术数知名故世所传竒异诡怪之事往往皆归之于朴如王铚黙记所载朴与周世宗微行中夜至五丈河旁见火轮小儿知宋将代周其事絶诞妄不可信而小説家顾乐道之宜作此书者亦假朴名以行矣然其间所引各书篇目大都皆宋以前本其综核数理剖晰义蕴亦多微中疑亦出自宋人非后来术士之妄谈也其书宋史艺文志不载诸家书目亦罕着録惟永乐大典颇散见其文虽间有缺脱而掇拾排比犹可得十之七八谨褒辑成编厘为六卷朴之名则削而不题以袪其伪焉乾隆四十六年九月恭校上

　　原序
　　至神无体妙万物以为体至道无方鼓万物以为用故浑沦未判一气湛然太极才分三才备位是以隂阳无私顺万物之理以生之天地无为辅万物之性以成之夫人生居天地之中虽禀五行之英为万物之秀者其形未兆其体未分即夙具其美恶蕴其吉凶故其生也天地岂容巧于其间哉莫非顺其世循其理辅其自然而已故夙积其善则赋其形美而福禄也素积其恶则流其质凶而处夭贱此其灼然可知其确然不易也是以古之贤圣察其人则观其形观其形则知其性知其性则尽知其心尽知其心则知其道观形则善恶分识性则吉凶显著且伏羲日角黄帝龙顔舜目重瞳文王四乳斯皆古之瑞相见之间降之圣人也其诸贤愚修短犹之指掌防毫丝末岂得逃乎故相论形神之术自此而兴焉其来极多其论至冗许负袁天纲陶隐居李淳风之后不可胜计然皆穷幽探得之至妙其或紊乱所说或异或同至使学者不能贯于一致余自穉嵗潜心于此考古验今无不征效遂特离林屋洞下山三载徧搜古今考之极者集成一家之书目之曰太清神鉴以其至大至明形无不鉴至清至莹象无不分然未足夺天地赋形之机亦可尽人之性形耳谨序

　　说歌　道为貌兮天与形黙授隂阳禀性情隂阳之气天地真【人伦风鉴云隂阳之气氛氲】化出尘寰几様人五岳四渎皆有神金木水火土为分君须识取造化理相逢始可论人伦贵人骨格定奇异防之乃为神仙邻若非古恠即清秀若非端正即停匀【人伦风鉴云丰姿不嫩仍端异】骨格洒落松上鹤【清】头角挺特真麒麟【古】森森修竹鎻流水【秀】峩峩恠石收闲云【恠】昆山片玉已琢出【端】南海明珠光照室【异】夭桃繁杏媚春华可怜容易摧风日【嫩】坐中初防似昻藏熟视稍觉无晶光语言泛泛失伦序举动碌碌多仓忙若人赋得此形相薄禄为官不久长坐中初见似尘俗熟视稍觉多清凉议论琅琅悉可听容止悠悠而细长若人赋得此形相高名美誉揽金章更防面部何气色数中惟有火多殃青多忧饶黑多病白多破财黄乃昌湛然沉静无瑕翳青云万里防翺翔富贵贫贱生处定但把形神来取正一部吉兮吉必生一部防兮凶必应部位吉凶各有主存神定意详观听妙理不过于五行当究五行之正性木瘦金方乃常谈水圆土厚何须竞不露不麄不枯槁三停大体求相称火形有禄终须破奔走贫寒多阻挫【人伦风鉴云惟有火形尖更露纵饶得禄终多破】虽因神秀暂荣华四十之上亦难过其余相法固非一天收地敛终无失气和神定最有常骨耸额寛根本实腰背端如万斛舟瞻视盼顾如星斗肉隐骨中骨隐体色隐神中神隐眸若人赋得此形相定知不是寻常流气宇汪洋有容物智量深逺多权谋动作令人不可料时通亦自为公侯易喜易怒属浅薄易骄易满属轻浮浅薄轻浮神不定一生自是常常忧欲知富贵何所致马面【人伦风鉴作马耳】牛头耸鼻梁有声有韵骨格清有頥有面含神光【人伦风鉴云起坐昂昂多神气】欲知贫者何所分面带尘埃眼目昏出语三言不辨两凹【人伦风鉴作凸】胷削背仍高臀赤脉纵横贯双眼杀人偷盗身无存人生具体皆相同贵贱相近有西东冲和而上主清轻认其清者宜高崇滞伏而下主重浊认其浊者皆凡庸清浊一分知贵贱贵贱不离清浊中大道无形故无相此理元来本至公【人伦风鉴云吉凶生于一念中】人能移恶归诸善自然可以消灾凶人能安分委天命自然可以济穷通【人伦风鉴云自然可以起凡庸叔敖隂徳故所劝上相相心今孰同】予作此诗真有理寄言贤者莫匆匆【人伦风鉴洞经同】又歌　受气成形三才俱备清则富贵浊则灾否但防贵人神清气爽眉目秀媚骨要坚隆骨肉相继骨谓之君肉谓之臣骨过于肉君过于臣此乃贵人长夀无迍龙骨吞虎生必豪富虎骨吞龙一世贫穷下短上长富贵吉昌上短下长遍走他乡面黑身白位至相国身黑面白卖尽田宅面麄身细一生富贵面细身麄贫困而孤额耸而隆不受贫穷额方面广有田有庄额骨而髙必为僧道额上有纹早年艰辛若是女子夫位难停额窄眉深卖尽资金额有伏犀不富则贵眉秀而清四海闻名眉如初月衣食不缺眉骨而髙长受波涛眉有旋纹父母不停八字眉分一生孤贫眉头相连夀短不延眉生纎毛上夀坚牢眉硬如锥晚嵗饥栖两眼藏神富贵髙名鱼尾插额位至相国睛色有黄为人不良若是红紫刀剑中亡斜视如流为性奸偷三角有光贼性难防四白羊睛杀子不停反视鬬睛为性不平睛露脸髙促夀不超鼻头圆红不受贫穷若要清贵年夀通隆鼻梁曲陆一生孤独鼻如鹰嘴啄人脑髓鼻梁横卖尽田园【人伦风鉴云破尽平生】鼻有纵纹尅子不停鼻头青紫晚年穷死口紫而方广买田庄【人伦风鉴云口如吹火孤寒独坐】口上有纹失约无成口薄而轻亲业如倾露齿结喉走遍他州耳有垂珠富贵双居【人伦风鉴云衣禄有余】轮廓皆成一世丰荣【人伦风鉴云耳内生毫夀永年髙耳生黒子多招是非】鼻梁耳返卖尽田产耳如箭翅贫穷破祖耳耸而朝富贵年髙髪踈而细位过两制发麄如麻贫穷可嗟女人髪拳尅夫连绵妇人髪黄遍走他乡手如噀血衣食不缺手软如绵衣食自然手纹如丝必为贤士手纹横生不聚资金纹如铁斧法死众覩【人伦风鉴云纹如戈矛法死不留】手上露筋一世艰辛声如破锣田产消磨声鸭声喑家计如倾声如大铙一世波涛舌上有纹牛马成羣舌如红莲广有田园舌短唇长晚嵗荒忙足下有纹大旺子孙足有纹一世清名足下黑子富贵贤士曲背驼腰子孙不超鸭步鹅行富贵家荣【人伦风鉴云富贵清名】行若蛇行亲业如倾蛇胷鹊筋【人伦风鉴云蛇骨鹤筋】贫穷贱人头大有肩富贵长年头大无肩晚嵗孤寒身大头小夀数极夭三停不直不为贵格南人北形富贵髙名北人南形产业如倾男人声雌破却家资女人声雄夫主不停眼下干枯定杀五夫齿露声雄杀夫贫穷更若有须不可同居形容如鬼杀夫不已面如哭形家业不成面上生靥夫位髙判形寒额尖夫位须死若是男子亦复如此人中黑子主养他子口上生者吐血而死泪堂生痣子孙难继黑子口傍是非难防兰堂生者富贵夀长眼下并纹子息难成纵理入口饿死不久法令入頥一生富贵纹入承浆夀限髙强令纹傍口财帛难守鼻凖有纹溺水而死骨怯神怯三犯须倾形重骨刚夀命延长三停隆直富贵无敌行步身斜破亲亡家脑有玉枕九十为定鼻梁不正四十絶命鼻曲唇掀夀夭不延【人伦风鉴云孤独如梦】肥人结喉夀短不留带杀双兼恶死居前有胷有背富贵无穷有背无胷晚嵗孤穷【人伦风鉴云有胷无背贫贱如流】声音如嘲贫贱不贵身小声雄位至三公身大声小早嵗折夭此法秘之如金如寳【人伦风鉴千字文同上】
