　　钦定四库全书　　　　　子部七
　　京氏易传　　　　　　　术数类四【占卜之属】提要
　　【臣】等谨案京氏易传三卷汉京房撰吴陆绩注房本姓李吹律自定为京氏字君明东郡顿邱人受易于焦延夀元帝时以言灾异得幸为石显等所嫉出为魏郡太守卒以谮诛事迹具汉书本传绩有易注已着録房所着有易传三卷周易章句十巻周易错卦十巻周易妖占十二卷周易占事十二巻周易守株三巻周易飞候九卷又六巻周易飞候六日七分八巻周易四时候四卷周易混沌四卷周易委化四巻周易逆刺占灾异十二巻易传积算法襍占条例一卷今惟易传存考汉志作十一篇文献通考作四巻均与此本不同然汉志所载古书卷帙多与今互异不但此编通考所谓四巻者以晁陈二家书目考之盖以杂占条例一卷合于易传三巻共为四卷亦不足疑惟晁氏以易传为即错卦杂占条例为即逆占灾异则未免臆断无据耳其书虽以易传为名而絶不诠释经文亦絶不附合易义上巻中卷以八卦分八宫每宫一纯卦统七变卦而注其世应飞伏游魂归魂诸例下卷首论圣人作易揲蓍布卦次论纳甲法次论二十四气候配卦与夫天地人鬼四易父母兄弟妻子官鬼等爻龙德虎形天官地官与五行生死所寓之类盖后来钱卜之法实出于此故项安世谓以京易考之世所传火珠林即其遗法以三钱掷之两背一面为坼两面一背为单俱面为交俱背为重此后人务趋防径以为卜肆之便而本意尚可考其所异者不以交重为占自以世为占故其占止于六十四爻而不能尽三百八十四爻之变张行成亦谓卫元嵩元包其法合于火珠林火珠林之用祖于京房陆德明经典释文乃于周易六十四卦之下悉注某宫一世二世三世四世游魂归魂诸名引而附合于经义误之甚矣乾隆四十六年十二月恭校上

　　【干下干上】干纯阳用事象配天属金与坤为飞伏居世【壬戌土癸酉金】易云用九见羣龙无首吉【纯阳用九之徳】九三三公为应肖干干夕惕之忧甲壬配外内二象【干为天地之首分甲壬入干位】积算起己巳火至戊辰土周而复始【吉凶之兆积年起月积日起时积时起卦入本宫】五星从位起镇星【土星入西方丽西北居壬戌为伏位】参宿从位起壬戌【壬戌在世居宗庙】建子起潜龙【十一月冬至一阳生】建已至极主亢位【四月龙见于辰阳极隂来吉去凶生用九吉】配于人事为首【干为首也】为君父【乾象坚刚天地之尊故为君父】于类为马为龙【天行运转不息】降五行颁六位【十二辰分六位升降以时消息吉凶】居西北之分野隂阳相战之地易云战于干【干为阳西北隂阳入隂二气盛必战】天六位地六气六象六包四象分万物隂阳无差升降有等【阴阳二十四律吕调矣】人事吉凶见乎其象造化分乎有无【故云变动不居周流六虚】六位纯阳隂象在中【阳中隂隂中阳】阳为君隂为臣阳为民隂为事阳实隂虚明暗之象隂阳可知【三五为阳二四为隂初上潜亢】水配位为福徳【甲子水是干之子孙】木入金乡居宝贝【甲寅木是干之财】土临内象为父母【甲辰土是干之父母】火来四上嫌相敌【壬午火是干之官鬼】金入金乡木渐微【壬申金同位伤木】宗庙上建戌亥干本位【戌亥干之位】阳极隂生【降入姤卦】八卦例诸
　　【巽下干上】姤隂爻用事金木互体天下风行曰姤姤遇也易曰隂遇阳【一隂初生阳气犹盛隂未为敌】与巽为飞伏元士居世【辛丑土甲子水】尊就卑【子孙与父母相代位】定吉凶只取一爻之象【多以少为贵】九四诸侯坚刚在上隂气处下易云系于金柅巽积隂入阳辛壬降内外象建庚午至乙亥【芒种小雪】积算起乙亥水至丙戌土周而复始【灾福之兆生乎五行升降也】五星从位起太白【太白在西居金位】井宿从位入辛丑【辛丑入土元上临母也】建午起坤宫初六爻易云履霜坚冰至建亥龙战于野【戍亥是干之位干伏本位必战积隂之地犹盛故战】配于人事为腹为母【坤顺容于物】于类为马易云行地无疆【此释一爻配坤象本体是干巽今賛一爻起隂假坤象言之】内巽为风干为天天下有风行君子以号令告四方【巽入也风入于坤皆动也故知天下有风动其物也】天风气象三十六【三十六节气降大风象】木入金为始【金纳木也】隂不能制于阳附于金柅易之柔道牵也五行升降以时消息隂荡降入遯【天山遯卦】
　　【艮下干上】遯隂爻用事隂荡阳遯金土见象山在天下为遯【遯退也】隂来阳退也小人君子汚隆契斯义也易云遯世无闷与艮为飞伏大夫居世建辛未为月【丙午火丙寅木】六二得应与君位遇建焉臣事君全身逺害【遯俟时也】建辛未至丙子隂阳遯去终而伏位【从六月至十一月也】积算起丙子至乙亥周而复始【火土同宫天与山遯】阳消隂长无専于败系云能消息者必専者败五星从位起太隂鬼宿入位降丙辰【丙午临元土】配于人事为背为手【艮为背手】于类为狗为山石内外升降隂阳分数二十八候【分隂阳进退】土入金为缓积阳为天积隂为地山所地髙峻逼通于天是隂长阳消降入否【隂逼阳去入天地否卦】
　　【坤下干上】否内象隂长【纯用隂事】天气上腾地气下降二象分离万物不交也小人道长君子道消【隂小人阳君子】易云否之匪人与坤为飞伏三公居世【乙卯木甲辰土】上九宗庙为应君子以俟时小人为灾【乙卯泰来】建壬申至丁丑隂气浸长【七月立秋至十二月大寒】积算起丁丑至丙子周而复始【金丑土同宫吉凶见矣】五星从位起嵗星【木星入卦用事】柳宿从位降乙卯【乙卯临三公】气分气候三十六【六六三十六积算吉凶】隂阳升降阳道消铄隂气凝结君臣父子各不迨及【隂荡阳来道行矣】易曰其亡其亡系于苞桑【苞桑则丛桑也】天地清浊隂薄【音搏】阳消天地盈虚与时消息危难之世势不可久五位既分四时行矣【君子当危难世独志难不可久立特处不改其操将及泰来】上九云否终则倾何可长也【否极则泰来】隂长降入于观【九四被隂逼入观卦】
　　【坤下巽上】观内象隂道已成威权在臣虽大观在上而隂道浸长与巽为飞伏诸候临世【辛未土壬午火】反应元士而奉九五【君位也】易云观国之光利用賔于王【臣道出于六四爻也】建癸酉至戊寅隂阳交伏【秋分至立春】积算起戊寅至丁丑周而复始【用金为首】金土火互为体五星从位起荧惑【火星入卦用事吉凶】星宿从位降辛未【星宿入诸侯宫木星同位】土木分气二十八【积算分配六位吉凶爻定数】隂阳升降定吉凶成败取六四至于九五成卦之终也易云观我生【我生即道也】又云风行地上【君子之徳风小人之徳草也】列象分爻以定隂阳进退之道吉凶见矣地上见巽积隂凝盛降入于剥【九五退隂入剥卦】
　　【坤下艮上】剥柔长刚减天地盈虚【建戌至建亥】体象金为本随时运变水土用事成剥之义出于上九易云硕果不食君子得舆小人剥庐【君子全得剥道安其位小人终不可安也】与艮为飞伏【丙子水壬申金】天子治世反应大夫建甲戌至己卯隂阳定候【寒露至春分】积算起己卯木至戊寅木周而复始【吉凶之兆见于有象】纯土配金用事五星从位起镇星【土星入卦】张宿从位降丙子【张宿入天子宫】金土分气三十六【积算六位起吉凶天地盈虚气候】易象云山附于地剥君子俟时不可茍变存身避害与时消息春夏始生天气盛大秋冬严杀天气消灭故当剥道已成隂盛不可逆阳息隂専升降六爻反为游魂荡入晋【积隂反入晋卦】
　　【坤下离上】晋隂阳返复进退不居精粹气纯是为游魂【为隂极剥尽阳道不可尽灭故返阳道道不复本位为归魂例入卦】金方以火土运用事与离为飞伏【己酉金丙戌土】诸侯居世反应元士建己卯至甲申隂阳继【春分立秋】积算起甲申金至癸未土周而复始【游魂取象配于正位吉凶同矣】五星从位起太白【卦配金星入用】翼宿从位降己酉金【翼宿北方入晋卦行事】二象分二十八运配金土积算气无差于晷刻吉凶列陈象在其中矣天地运转气在其中矣乾道变化万物通矣【干分八卦至大有复卦】六爻交通至于六卦隂阳相资相返相尅相生至游魂复归本位为大有故曰火在天上大有为归魂卦定吉凶配人事五行象干为指归地【凢八卦分为八宫每宫八卦八八六十四卦定吉凶配人事天地山泽草木日月昆虫包含气足矣】
　　【干下离上】大有卦复本宫曰大有内象见干是本位【八卦本从干宫起至大有为归魂】纯金用事与坤为飞伏【甲辰土乙卯木】三公临世应上九为宗庙建戊寅至癸未【立春正月至大暑时也】积算起癸未土至壬午火周而复始【吉凶与乾卦同用】五星从位起太隂【太隂水星入卦用事】轸宿从位降甲辰【二十八宿分轸星入大有卦用事行度吉凶可见】金土分象三十六候配隂阳升降六位相荡返复其道【复归本位也】吉凶度数与乾卦同分六五隂柔为日照于四方象天行健【六龙御天】少者为多之所宗六五为尊也【柔处尊位以柔履刚以隂处阳能柔顺于物万物归附故曰照于四方】易曰火在天上大有【离为火为日故曰大有】隂阳交错万物通焉隂退阳伏返本也乾象分荡八卦入大有终也干生三男次入震宫八卦【干生三男坤生三女阳以阳隂以隂求竒耦定数于象也】
