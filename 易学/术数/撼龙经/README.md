　　撼龙经　疑龙经　法倒杖　术数类三【相宅相墓之属】提要
　　【臣】等谨案撼龙经一卷疑龙经一卷法倒杖一卷旧本题唐杨筠松撰筠松不见于史传惟陈振孙书録解题载其名氏宋史艺文志则但称为杨救贫亦不详其始末惟术家相传以为筠松名益赣州人掌灵台地理官至金紫光禄大夫广明中遇黄巢犯阙窃禁中玉函秘术以逃后往来于处州无稽之谈盖不足尽信也然其书乃为世所盛传撼龙经专言山垅落脉形势分贪狼巨门禄存文曲防贞武曲破军左辅右弼九星各为之説疑龙经上篇言干中寻枝以关局水口为主中篇论寻龙到头看面背朝迎之法下篇论结穴形势附以疑龙十问以阐明其义法则专论穴有倚盖撞黏诸説倒杖分十二条即上説而引伸之附二十四砂法亦临穴时分寸毫厘之辨案书録解题有疑龙经一卷辨龙经一卷云呉炎録以见遗皆无名氏是此书在宋并不题筠松所作今本不知何据而云然其撼龙之即辨龙与否亦无可考证然相传已久所论山川之性情形势颇能得其要领流传不废亦有以也旧本有李国本注并所附各图庸陋浅俗了无可取今并加刋削不使与本文相溷焉乾隆四十六年十月恭校上

　　钦定四库全书
　　撼龙经　　　　　　　　唐　杨筠松　撰
　　须弥山是天地骨中镇天地为巨物如人背与项梁生出四肢龙突兀四肢分出四世界南北东西为四西北崆峒数万程东入三韩隔杳防惟有南龙入中国胎宗孕祖来竒特黄河九曲为大膓川江屈曲为膀胱分肢擘脉纵横去气血勾连逄水住大为都邑帝王州小为郡县君公侯其次偏方小镇市亦有富贵居其中大率龙行自有真星峯磊落是龙身髙山须认星峯起平地龙行别有名峯以星名取其类星辰下照山成形龙神二字寻山脉神是精神龙是质莫道髙山方有龙却来平地失真踪平地龙从髙脉发髙起星峯低落穴高山既认星峯起平地两傍寻水势两水夹处是真龙枝叶周回中者是莫令山反枝叶散山若反兮水散漫外山百里作罗城此是平洋龙局段星峯顿伏落平去外山隔水来相顾平中仰掌似凹窠隠隠微微立丘阜倾从丘阜覔凹窠或有勾夹如旋螺勾夹是案螺是穴水去明堂聚气多四傍绕防如城郭水绕山还聚一窝霜降水涸寻不见春夏水髙龙背现此是平洋看龙法过处如丝或如线髙水一寸即是山低水一寸水回环水纒便是山纒様纒得真龙如仰掌窠心掌里或乳头端然有穴明天象水绕山纒在平坡逺有冈陵近有河只爱山来抱身体不爱水返去从他水抱应知山来抱水不抱兮山不到莫道髙山龙易识行到平洋失踪迹藕断丝连正好寻退卸愈多愈有力髙龙多下低处藏四没神机便寻得祖宗父母数程遥误得时师皆不识凡到平地莫问踪只观环绕是真龙念得龙经无眼力万卷真藏也是空
　　北辰一星中天尊上相上将居四垣天乙太乙明堂照华葢三台相后先此星万里不得一此龙不许时人识识得之时不用藏留与皇朝镇家国请从垣外论九星北斗星宫系几名贪巨武星并辅弼禄文亷破地中行九星人言有三吉三吉之余有辅弼不知星曜定锱铢祻福之门敎君识
　　贪狼顿起笋生峯若是斜枝便不同斜枝侧顶为破靣尖而有脚号乗龙脚下横拖为劒文武功名从此辩横看是顶侧是峯此是贪狼出阵龙侧靣成峯身直去不是为朝便不住
　　莫来此处认髙峯道是武在其中亦有髙峯是武武落处四兽聚聚处方为龙聚星四兽不顾只成空空亡龙上莫寻穴纵然有穴易歇灭
　　或为关峡似龙形正身潜在峡中行时师多向峡中覔不识真龙断续情贪狼自有十二様尖圆平直小为上欹斜侧岩倒破空祻福轻重自不同欹侧似斜斜似侧平似乗龙侧似直贪狼似巨倒似空空似虚岩即似石问君来此如何观我道贪狼非一般欹是崩崖破是折斜是边有边不同侧是靣尖身直去空是岩穴多玲珑倒是飞峯偏不正七者未是贪狼龙平地卓然顿起笋此是尖狼本来性圆无欹侧四靣同平若卧蚕在髙顶直如决脊引绳来小似笔头揷髙塔五者方为贪正形吉凶祻福要详明
　　火星要起廉贞位生出贪狼由此势若见火星动熖时看他踪迹落何处此龙不是寻常贵生出贪狼向亦竒火星若起廉贞位落处须寻一百里中有贪狼小小峯有时回顾火星宫世人只道贪狼好不识廉贞是祖宗贪狼若非廉作祖为官也不到三公髙山顶上如平掌中分细脉如蛇様贵龙多是穿心出富龙只从傍生上髙山如帐后靣遮帐里微微似斜舞下来如防尾此是贪狼上岭蛇舞下来伸鹤颈此是贪狼下岭蛇上岭解生朱紫贵下岭须为朽腐家
　　大山特起小为贵小山忽起大为势髙低大小断续行此是贪狼真骨气大抵九星有种类生子生孙巧相似剥换方知骨气真剥换不真皆不是一剥一换大生小从大剥小最竒异剥换退卸见真龙小峯依旧贪狼起剥小如人换好裳如蝉退殻蚕退筐或从大山落低小或从高峯落平洋退卸剥换成几段十条九条乱了乱中有一条却是真若是真时断了断乱山回抱在靣前不许一条出外边只有真龙在帐内乱山在外却为纒此龙多从腰里落回转余枝作城廓城廓弯环生捍门门外罗星当腰着罗星要在罗城外此与火星常作案火星龙始有罗星若是罗星不居内居内名为抱养瘝又为病跟堕胎山罗星若生罗城口城口皆为玉笋班罗城恰似城墙势龙在城中聚真气罗星借在城阙间时师唤作水口山欲识罗星真妙诀一边枕水一边田田中有骨胍相连或为顽石焦土间此是罗星有余气卓立为星在水边贪巨罗星方与尖辅弼武曲员匾眠禄存廉贞多破碎破军尖破最为害只有尖圆方匾星此是罗星得正形忽然四靣皆是水两山环合欝然青罗星亦自有种类浪説罗星在水边
　　巨门尊星性端荘才离祖宗即高昻星峯自与众星别不尖不圆其体方髙处定为顿笏様但是无脚生两傍如此星峯止一二方冈之下如驱羊方冈或如四角帐帐中出似飞扬飞扬要得穿帐去帐中两角随身张枝叶不多关峡少却有防卫随身傍旌节来拥防旌节之峯多是双更有刀剑同防送刀剑送后前圆冈离踪断处多失胍抛梭马迹蛛丝长梭中自有丝不断蜂腰过处多趋跄
　　自是此星性尊贵防送此星来就体每逄跌断过处时两傍定有衣冠吏衣冠之吏似圆峯两傍有脚卫真龙若是独行无防卫定作神祠佛道宫平行穿珠行数里忽然又作方峯起方峯直去如桥扛背长颇类平尖贪平尖贪狼如一字生在山顶如卧蚕武曲倒从身中出贪狼直去如僧叅夹辅防龙次第列正龙在内左右凾此龙住处无髙垅间生窝穴隐深潭独在山峡中间者穴落髙冈似草庵四围要髙来朝防前案朝迎亦髙舞却作髙穴似人形按敛端严似真武
　　此龙若行三十里内起方峯止三四峯峯端正方与长不肯欹斜失尊体峯上忽然生折痕此与廉贞何以异凡起星辰不许斜更嫌生脚照他家端峯若生四花穴花穴端严要君别真龙直去向前行四向谩成龙虎穴此是武曲钳峡来间气来此偶生峡此龙误了几多人定来此处説真形要説四花穿心过但看防卫不曾停尊星自有尊星体方正为屏将相位巨门行龙少鬼刼葢缘两傍多罗列水界分处夹龙行不肯单行走空缺水界分及乱生枝枝叶虽多夹水随防龙亦自有背靣背后如壁靣平夷平夷便是贴龙体龙过之时形怪异不起尖圆即马旗攒劒繙龙归此地防卫纒绕如打团重重包裹外山归至令巨门少关峡防送无容左右离明堂断定无斜防横案重重拜舞低平贪覆巨圆武曲尖圆方整不能齐三星尖圆方整处向此辩别无狐疑识龙须识辨疑处识得真龙是圣师
　　禄存之形如顿鼔下生有脚如瓠瓠头前有小峯此是禄存禄处大如螃蠏小蜘蛛此是禄存杀处杀中若有横磨敛此是权星先出武
　　大龙大峡百十程宝殿龙楼去无数【峡口防平曰殿】忽履仁等入长垣【长垣如城】万仞不围君莫顾痴师偷眼傍睥晲晓者黙然佯不睹若然尖脚乱如茅唤作蚩尤旗爪距【天上有蚩尤旗星】小圆禄围本身将相公侯出方虎
　　大抵星辰嫌破碎不抱本身多作怪端正龙神须无破丑恶龙神多破败怪形异穴出凶豪杀戮平民终大壊草头作乱因此山赤族诛夷偿命债只縁龙上有欃枪贼旗倒仄非旌幢旌幢对对端正立独立欹仄名欃枪顿鼓微方似武曲武曲端正下无足有足周围真禄存圆尽方为武曲尊龙家最要仔细辨疑似乱真分背靣背似靣非岂有真此是禄存大移转凹处是靣凸是背作穴分金过如线凡看星辰看转移转移须敎母顾儿枝分别有真种忽作蔓无东西十里半程无冈岭平阳砂碛烟尘迷到处君须看水势水势莫问江与溪只要两源相夹出交外结重重围禄存好处落平洋大作方州小镇县坪中时复乱石生或起横山或梭靣此处或有辅弼形辅弼无枝禄生辨禄是帝车第二星也主为文也主兵
　　九星行龙皆要禄最要夹贪兼巨轴或从武曲左右起此等贵龙看不足若逢此星逺寻穴莫向髙山寻促局若遇九星相夹行只分有足并无足燕云下岭出九关中禄存三吉山髙山峡里多尖秀也有圆禄生孱岩君看山须分种类乱指横山作正班禄破二星形无数也有正形落低处也有低形上垅头杂乱分形君莫误形在髙岭为髙形山顶上生禄存星形在平洋山卓立顶矮脚手亦横平顶上生形顶必正平地生形脚乱行请君看我细排列祻福皆从龙上生
　　第一禄存如顿鼔脚手对对随身去平行有脚如劒芒旌节幡幢排次序此等星辰出大江中有小贪并小巨辅弼侍从左右生隔岸山河逺相顾此是龙身作州县雄据十州并一路忽然诸山作垣局更求吉水为门户若得吉水为门户万水千山不须做
　　第二禄存如覆釡脚尖如防周回布有脚方为真禄存无脚方为禄堆巨此星定是有威权白手成家积巨富第三禄存鹤爪布两短中长龙出露出露定为低小形隠隠前行忽蹲踞有穴必生龙虎巧丑陋穴形龙不住第四禄存肋扇具脚手又似扛丝势此龙只好结神坛别有星峯生秀气
　　第五禄存如悬鹑破碎箕帚折无数此星便是平行星星平生枝自顶分此龙只去平中作桡棹回来斩关做髙山大峡开三路
　　第六禄存落平洋势如巨浪横开张他星亦有落平者此星平地亦飞扬脚摆时复生巨石石色只是黒与黄两傍请看随龙峡长短大小宜推详防龙转时看他落落处当随水斟酌右转皆右不参差左转皆左无驳杂朝迎指正真穴形左右髙低君莫错禄存鬼形如披髪虽曰众多势如掠
　　第七禄存如长蛇左右无防无拦遮此龙目作贵龙从枕在水边自横斜
　　第八禄存在髙顶如载兠鍪有肩领渐低渐小去作穴定作窝钳极端正此龙号为八贵龙捉穴真时最昌盛第九禄存如落花片片水夹砂不作蛟潭为鬼穴定作罗星水口遮
